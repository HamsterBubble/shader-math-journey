Ray Marching 的核心思想

"射线步进"：从相机发出一条射线，沿着射线一步步往前走。每走一步，用 SDF 查询"离最近物体有多远"。走的步长 = 该距离（因为 SDF 保证在这个范围内不会穿过任何物体）。当距离足够小时 → 碰到物体了！

算法框架

for (int i = 0; i < MAX_STEPS; i++) {
  vec3 p = ro + rd * t;     // 当前位置
  float d = sceneSDF(p);    // 到最近物体的距离
  if (d < 0.001) break;     // 足够近 = 碰到了
  t += d;                   // 安全步进
  if (t > MAX_DIST) break;  // 走太远 = 什么都没碰到
}

关键组件

  • ro（Ray Origin）：相机位置
  • rd（Ray Direction）：从 UV 计算的射线方向
  • sceneSDF：3D 场景的 SDF（可以复用 2D 学到的所有图元！）
2D → 3D 的跨越：
你在前面学的所有 SDF 知识——布尔运算、域重复、平滑合并——在 3D 中完全适用。只需把 vec2 换成 vec3，把 2D 图元换成 3D 版本（球、方块、圆柱...）。这就是 SDF 的美：维度无关。
💡 法线计算
碰到表面后，用中心差分算法（和 Stage 6 一样）计算 3D 法线：
normal = normalize(vec3(
  sdf(p+e.xyy)-sdf(p-e.xyy),
  sdf(p+e.yxy)-sdf(p-e.yxy),
  sdf(p+e.yyx)-sdf(p-e.yyx)))
有了法线，就能做完整的 Phong 光照了！
