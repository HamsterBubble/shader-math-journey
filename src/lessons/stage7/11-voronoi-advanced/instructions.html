🔮 高级 Voronoi 纹理

超越 F1 距离

基础 Worley 噪声用"最近点距离 F1"。但 Voronoi 系统还提供 F2（第二近），而 F2 - F1 揭示了细胞的边界。

F1 和 F2 的艺术

  • F1：基础细胞噪声 → 生物纹理
  • F2 - F1：细胞边线（裂纹、龟裂、蜻蜓翅膀）
  • F2：反转的细胞 → 圆角方块图案
  • F1 × F2：有趣的复合纹理
最近中心 ID = 颜色
保存最近点的坐标或 cell ID，用它做 hash → 每个细胞不同的随机颜色。这就是彩色 Voronoi 图。
💡 带动画的 Voronoi
让种子点位置加上 sin(u_time + hash) → 细胞缓慢蠕动。非常适合有机/生物效果。
