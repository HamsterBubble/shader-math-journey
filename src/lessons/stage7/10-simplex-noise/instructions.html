Value Noise 的升级版

Value Noise 在正方形网格上插值——会产生明显的轴向偏好。Simplex Noise 用单纯形网格（2D=三角形，3D=四面体），效果更均匀、计算量更少。

Simplex vs Value vs Perlin

  • Value Noise：网格顶点存随机值，双线性插值 → 方块感
  • Perlin Noise：网格顶点存随机梯度，Hermite 插值 → 轴向伪影
  • Simplex Noise：三角形网格 + 梯度 → 最少伪影，O(N) 复杂度
核心思路：
1. 用倾斜变换将正方形网格→三角形网格
2. 找到点所在三角形的 3 个顶点
3. 对每个顶点：梯度 · 偏移向量 × 衰减核
4. 求和
倾斜常数的由来
K1 = (√3 - 1) / 2 ≈ 0.366 → 正方形→三角形的倾斜系数
K2 = (3 - √3) / 6 ≈ 0.211 → 反向倾斜系数
