运用本课程学到的所有技巧，创造一个深邃而生动的程序化星空——你的毕业作品。

目标效果（实时）

星空的视觉拆解

  • 背景：深蓝/深紫的渐变，微弱的星云（FBM + 调色板）
  • 恒星：不同大小和亮度的亮点（Worley F1 或网格哈希）
  • 闪烁：每颗星有独立的闪烁节奏（相位偏移的 sin）
  • 流星/银河：加分装饰元素

推荐思路

星星生成：多层分辨率的网格，每层密度和大小不同

// 每层用不同的网格大小
for 3 layers:
  cellID = floor(uv * scale)
  hash(cellID) → 位置偏移 + 大小 + 亮度
  dist = length(cellUV - starPos)
  star = exp(-dist * sharpness) × brightness

星云背景：FBM + 域扭曲 + 低饱和度调色板

闪烁：brightness *= 0.7 + 0.3 * sin(u_time * speed + hash(cellID) * 6.28)

如果你能独立完成这个效果，说明你已经掌握了 Shader 数学的核心！
💡 层次感是关键
好的星空不是"一堆亮点"，而是有清晰的前/中/背景层次：
● 远处：很多很暗很小的星（细网格、低亮度）
● 中间：中等数量和亮度的星
● 近处：少量很亮、有光晕的大星
