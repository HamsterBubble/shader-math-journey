🌀 域扭曲 (Domain Warping)

用噪声来扭曲噪声

域扭曲是 Shader 中最强大的过程化纹理技巧之一：用一个函数的输出来偏移另一个函数的输入坐标。

result = fbm(p + fbm(p + fbm(p)))

每一层嵌套都增加了一层"扭曲"，让原本规则的噪声变得极其有机和自然。

为什么看起来这么"活"？

  • 第一层 fbm → 基础的随机起伏
  • 用 fbm 偏移坐标 → 起伏被"弯曲"，出现类似风吹、流动的纹理
  • 再嵌套一层 → 弯曲的弯曲，出现旋涡、烟雾般的复杂结构

控制参数

  • 偏移强度：fbm(p + fbm(p) * strength) → strength 越大扭曲越剧烈
  • 时间动画：fbm(p + fbm(p + u_time * speed)) → 缓慢演变的有机运动
  • 不同偏移方向：x 和 y 用不同的 fbm → 更自然的扭曲
这就是程序化大理石/木纹/云/岩浆的秘密：
几乎所有 Shadertoy 上的"有机"纹理——大理石纹路、岩浆流动、热气升腾——核心技巧都是域扭曲。
IQ 的经典文章 "warping by noise" 详细解释了这个技巧，被 Shader 社区奉为圣经。
💡 性能考量
嵌套 fbm 意味着每个像素要计算大量噪声采样：
● 1 层 fbm (4 octaves) = 4 次噪声采样
● 2 层嵌套 = 4 + 4 = 8 次
● 3 层嵌套 = 4 + 4 + 4 = 12 次
在移动端要注意层数和 octave 数的平衡。

2. 增加扭曲强度看效果变化

4. 三层嵌套 + 调色板 → 大理石/火焰/星云
