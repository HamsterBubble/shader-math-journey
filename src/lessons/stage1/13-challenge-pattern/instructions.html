分解思路

这个效果看起来复杂，但其实只需要本阶段学过的函数。让我们逐步拆解：

第一步：创建网格

用 fract(uv * N) 把屏幕分成 N×N 格，同时用 floor(uv * N) 获取每格的编号：

  • vec2 cellUV = fract(uv * N) - 0.5; — 每格的局部坐标，中心在 (0,0)
  • vec2 cellID = floor(uv * N); — 格子编号，用来区分不同格子

第二步：画脉动环

在每个格子内画一个半径随时间变化的环：

  • 计算到格子中心的距离：float d = length(cellUV);
  • 半径随 sin 脉动：float radius = 0.3 + 0.1 * sin(u_time + phase);
  • 画环：abs(d - radius) < thickness 用 smoothstep 柔化

第三步：每格不同的相位

如果所有格子同步脉动，效果很无聊。用 cellID 创造不同的时间偏移：

  • float phase = (cellID.x + cellID.y) * 0.5; — 对角线波浪
  • 或用伪随机：fract(sin(dot(cellID, vec2(12.9898, 78.233))) * 43758.5453)

第四步：加颜色和点缀

  • 用 cellID 或距离值驱动调色板
  • 中心加一个小圆点增加层次感
💡 挑战心法
● 先让一个格子工作，再加 fract 重复
● 先用硬 step，效果对了再换 smoothstep 柔化
● 先静态，对了再加 u_time 动画
一步步来，每步都能看到即时反馈！
