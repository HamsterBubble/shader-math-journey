🔄 fract() — 重复的魔法

从地板砖说起

走进一间铺满瓷砖的房间，每块砖的花纹完全一样。这就是 fract() 在做的事：把空间切成重复的小格子，每格里的坐标回到 0~1。

数学原理

fract(x) = x - floor(x)

floor(x) 取整数部分（向下取整），fract(x) 只保留小数部分。

  • fract(0.7) = 0.7
  • fract(1.3) = 0.3
  • fract(2.9) = 0.9

当 x 从 0 连续增长到 3 时，fract(x) 会在 0~1 之间循环 3 次，像锯齿波。

空间重复的关键公式

核心公式：fract(uv * N)
先把 UV 坐标放大 N 倍，再取小数 → 空间被分成 N × N 个格子，每个格子内的坐标重新从 0~1。

这意味着你只需要画一个格子里的图案，它会自动平铺整个画面！

fract 与 floor 是好搭档

当你用 fract() 得到格内坐标时，常常也需要知道"我在第几个格子"：

  • fract(uv * N) → 格内坐标（0~1）
  • floor(uv * N) → 格子编号（整数）

格子编号可以用来：给每个格子不同的颜色、不同的动画相位、不同的随机种子……这是很多高级效果的基础。

⚠️ 常见陷阱：fract 后的坐标中心在 (0.5, 0.5) 而不是 (0, 0)。如果你要画一个居中的圆，需要减 0.5：vec2 p = fract(uv * N) - 0.5;

🎯 动手试试

1. 改 3.0 → 5.0, 8.0（更多格子，图案更密）

2. 改圆的半径 0.3，观察每个格子里的圆大小变化

3. 只水平重复: uv.x = fract(uv.x * 5.0);（保持 uv.y 不变）

4. 用 floor(uv * N) 给不同格子不同的颜色
