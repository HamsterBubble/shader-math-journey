〰️ smoothstep() — S 曲线过渡

🎨mix 混合原理

从"硬切"到"柔和"

step() 是一把刀，非黑即白。但现实世界很少有这么锐利的边界——阴影有虚化、颜色有渐变、边缘有抗锯齿。smoothstep() 就是把刀变成了软刷。

数学原理

smoothstep(a, b, x)
  • 当 x ≤ a → 返回 0.0
  • 当 x ≥ b → 返回 1.0
  • 当 a < x < b → 返回 0~1 之间的平滑过渡值

这个"平滑"不是简单的线性插值。smoothstep 内部用了一个 Hermite 多项式：

t = (x - a) / (b - a)
result = t² × (3 - 2t)

这条 S 形曲线在两端的斜率为 0（"起步慢，结束也慢"），中间加速，看起来非常自然。

a 和 b 的距离决定"柔软度"

  • smoothstep(0.45, 0.55, x) → 窄范围，过渡很快，接近 step
  • smoothstep(0.2, 0.8, x) → 宽范围，大面积柔和过渡
  • smoothstep(0.499, 0.501, x) → 极窄，几乎等于 step
smoothstep 是 Shader 世界的"万能柔化剂"：
● 圆的边缘 → smoothstep(r-0.01, r+0.01, dist) = 抗锯齿圆
● 颜色过渡 → smoothstep(0.3, 0.7, uv.x) = 柔和渐变
● 光照衰减 → 1.0 - smoothstep(0.0, radius, dist) = 柔和光晕
💡 step vs smoothstep vs mix
● step — 硬切（0 或 1，无中间态）
● smoothstep — 平滑切（S 曲线过渡）
● mix — 线性混合（直线过渡）
它们是 Shader 的三大"过渡工具"，会贯穿你的 Shader 生涯。
