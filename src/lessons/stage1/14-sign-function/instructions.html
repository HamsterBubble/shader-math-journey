三值判断器

sign(x) 是最简单的分类器——它只回答一个问题："这个数是正的、负的、还是零？"

sign(x) = -1 (x < 0) | 0 (x = 0) | +1 (x > 0)

为什么需要 sign？

  • 方向判断：sign(uv.x) — 左半边=-1，右半边=+1
  • 翻转效果：value * sign(uv.x) — 让效果在中线两边镜像反转
  • 提取方向：sign(velocity) — 只关心运动方向，不关心速度大小

sign 与 abs 的关系

一个重要恒等式：
x = sign(x) × abs(x)
任何数都可以分解为"方向"（sign）和"大小"（abs）。这在向量数学中有深远影响。
💡 用 sign 替代 if
需要"x 为正时加 offset，为负时减 offset"？
不要用 if：result = x + sign(x) * offset;
无分支，GPU 友好。
