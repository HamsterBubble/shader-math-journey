📏 abs() 绝对值 与 mod() 取模

这两个函数在 Shader 中无处不在：

abs(x) — 绝对值

  • 把负值"折叠"成正值：abs(-3.0) = 3.0
  • 几何含义：沿轴做镜像！abs(uv.x) 让左右对称
  • 常用于 SDF：abs(distance) - thickness 画环

mod(x, y) — 取模

  • mod(5.0, 3.0) = 2.0（除以 3 的余数）
  • 等价于 x - y * floor(x/y)
  • 和 fract() 的关系：fract(x) = mod(x, 1.0)
  • 用于创建重复条纹或交替网格
关键洞察： mod(x, 2.0) < 1.0 可以判断奇偶条纹——这是棋盘格的基础。

🎯 动手试试

1. 注释掉 abs()，看对称性消失

2. 改 mod 的模数（2.0 → 3.0）看条纹密度变化

3. 用 mod(cellID.x + cellID.y, 2.0) 做棋盘格
