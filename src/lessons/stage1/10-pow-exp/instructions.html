📈 pow() 幂函数与 exp() 指数衰减

📐坐标居中原理⭕SDF 基础

控制"弯曲"的程度

前面学的 mix 和 smoothstep 都是"按比例过渡"，但有时你需要更精细的控制：让过渡在开始时很慢，然后突然加速——或者反过来。pow() 和 exp() 就是控制这种非线性变化的工具。

pow(x, n) — 幂函数

x 的 n 次方。当 x 在 0~1 范围内时，n 的值决定了曲线的"弯曲度"：

  • pow(x, 1.0) — 直线（线性，就是 x 本身）
  • pow(x, 2.0) — 抛物线（开始慢、后面快 = "ease-in"）
  • pow(x, 0.5) — 平方根（开始快、后面慢 = "ease-out"）
  • pow(x, 5.0) — 极端加速，只有靠近 1 时才亮

pow 在光照中的角色

Phong 高光模型的核心公式：pow(dot(R, V), shininess)

  • shininess = 8 → 蜡质柔和光
  • shininess = 64 → 金属锐利光
  • shininess = 256 → 镜面反射

指数越高，只有"正对"反射方向的很小区域才有亮点。

exp(x) — 自然指数衰减

exp(-x) 是大自然最喜欢的衰减方式——光的散射、声音的消减、热量的传导都遵循指数衰减。

  • exp(-x) — 距离越远越弱。光晕、雾效的基础
  • exp(-x*x) — 高斯曲线（钟形）。模糊算法的数学基础
  • 1.0 - exp(-x*k) — 渐近趋近 1.0，k 控制速度。常用于"充能"效果
Gamma 校正 — pow 的最重要应用
显示器不是线性显示亮度的：物理上 0.5 的亮度在屏幕上看起来约等于 0.73。
为了补偿这个非线性，我们对最终颜色做 Gamma 校正：
finalColor = pow(color, vec3(1.0/2.2))
这一行代码会让你的 Shader 看起来专业度提升 50%。
💡 快速记忆
● pow → 控制曲线"弯不弯"，n>1 加速，n<1 减速
● exp(-x) → 自然衰减，越远越弱
● exp(-x*x) → 钟形曲线，离中心越远越弱

2. 改 exp 的衰减系数看光晕大小变化
