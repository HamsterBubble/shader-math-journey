🔷 min() 和 max() 的几何含义

不只是"取较小/较大值"

在普通编程中，min/max 只是比较两个数。但当这两个数是 SDF 距离值时，min 和 max 就变成了强大的几何布尔运算器——它们能合并、切割、雕刻形状。

max(a, b) — 交集

取较大值 = "必须同时满足两个条件的区域"：

  • 只有当两个 SDF 都 ≤ 0（都在内部）时，max 结果才 ≤ 0
  • 几何含义：两个形状重叠的部分
  • 经典应用：max(abs(uv.x) - 0.3, abs(uv.y) - 0.2) → 矩形
  • 矩形的本质就是"x 在范围内 AND y 在范围内"

min(a, b) — 并集

取较小值 = "只要满足任一条件的区域"：

  • 只要任一 SDF ≤ 0，min 结果就 ≤ 0
  • 几何含义：把两个形状都画出来
  • 例如：min(circleSDF, boxSDF) → 圆和方块同时出现

max(a, -b) — 差集（挖洞）

这个稍微不直观：取反某个 SDF 意味着"接翻内外"，然后和另一个取交集 = 从一个形状中挖掉另一个。

  • max(bigCircle, -smallCircle) → 大圆减去小圆 = 环形（甜甜圈）
  • max(box, -circle) → 方块上挖一个圆孔
这就是 CSG（构造实体几何）！
3D 建模软件中的"布尔运算"（合并、相交、相减）在 SDF 中只需 min/max 就能实现。
● Union（并集） = min(a, b)
● Intersection（交集） = max(a, b)
● Subtraction（差集） = max(a, -b)
所有复杂形状都可以分解为简单形状的 min/max 组合。
💡 平滑布尔运算
直接 min/max 会产生锐利的接缝。在实际项目中，常用"平滑最小值"：
smin(a, b, k) 让两个形状的交界处柔和过渡（像果冻粘在一起）。
这是 Metaball（融球）效果的数学基础，Stage 5 会详细学习。
