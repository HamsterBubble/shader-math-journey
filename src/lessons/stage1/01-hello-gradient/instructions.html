🎨 你好，渐变！

Shader 是什么？

Shader（着色器）是运行在 GPU 上的小程序。

它的工作很简单：对屏幕上的每一个像素，计算一个颜色。

输入：像素坐标 gl_FragCoord.xy
输出：颜色 gl_FragColor = vec4(r, g, b, a)

UV 坐标

我们把像素坐标归一化到 0~1，这样代码和分辨率无关：

vec2 uv = gl_FragCoord.xy / u_resolution.xy;

  • uv.x = 0 最左 → uv.x = 1 最右
  • uv.y = 0 最下 → uv.y = 1 最上

🎯 动手试试

1. 把 uv.x 改成 uv.y → 竖向渐变

2. 第三个分量改成 1.0 → 加蓝色

3. 试试 vec3(uv.x, uv.y, 0.5)
