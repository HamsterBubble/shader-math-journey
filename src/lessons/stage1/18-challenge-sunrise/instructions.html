🏆 挑战：日出渐变

用 mix、smoothstep、exp 和 SDF 创造一幅日出风景。

目标效果（实时）

分步提示

  1. 天空：用多层 mix 做暖到冷的垂直渐变
  2. 太阳：圆形 SDF + smoothstep 硬边
  3. 光晕：exp(-distance * k) 柔和衰减
  4. 地面：step 或 smoothstep 在底部画暗色地面
  5. 动画：让太阳随 u_time 缓慢升起
💡 颜色参考
天空顶部 → 深蓝 (0.1, 0.1, 0.35)
天空中部 → 橙红 (0.8, 0.4, 0.2)
天空底部 → 金黄 (1.0, 0.6, 0.1)
太阳 → 亮白暖 (1.0, 0.95, 0.8)
