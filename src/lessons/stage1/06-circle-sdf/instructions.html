什么是 SDF？

SDF（Signed Distance Function，有符号距离函数）是 Shader 中画图形的核心方法。它的思路出奇地简单：

对于空间中的每一个点，计算它到图形边界的距离。

  • 距离 < 0 → 点在图形内部
  • 距离 = 0 → 点刚好在边界上
  • 距离 > 0 → 点在图形外部

"有符号"就是指：内部取负值，外部取正值。一个数字就能告诉你"在哪边、离多远"。

圆的 SDF

圆是最简单的 SDF，因为"到圆心的距离"就是现成的：

sdf = length(p - center) - radius
  • length(p - center) = 点到圆心的距离
  • 减去 radius → 内部变负、边界为零、外部为正

从距离到颜色

有了 SDF 值，怎么画出图形？最简单的方法：

  • step(0.0, -sdf) → 硬边圆（内部白、外部黑）
  • smoothstep(0.01, -0.01, sdf) → 柔边圆（抗锯齿）
  • 直接用 sdf 值映射颜色 → 看到距离场的"等高线"
为什么 SDF 如此强大？
因为它把复杂的几何问题简化为简单的数学运算：
● 并集（画多个图形）= min(sdf1, sdf2)
● 交集（取重叠区域）= max(sdf1, sdf2)
● 描边（只画轮廓）= abs(sdf) - thickness
● 圆角 = sdf - roundness
后面你会经常用到这些操作！
💡 SDF 的美学
直接可视化 SDF 值（如用颜色梯度映射 sdf）会得到非常漂亮的"等高线"效果，这在 debug 和艺术创作中都极为有用。
