🌊 sin() — 波浪函数

大自然的振荡器

水波、声波、光波、心跳、四季更替……自然界充满了周期性振荡。而 sin() 函数，就是描述这种"来回摆动"的数学语言。

sin 的核心性质

sin(x) 的输出在 -1 ~ 1 之间永远来回振荡，一个完整周期的长度是 2π（≈ 6.28）。

  • sin(0) = 0 → 起始点
  • sin(π/2) = 1 → 最高点
  • sin(π) = 0 → 回到零
  • sin(3π/2) = -1 → 最低点
  • sin(2π) = 0 → 一个完整周期结束，重新开始

三大控制参数

通用波动公式：

y = A × sin(F × x + P)
  • 振幅 A（Amplitude）：控制波的高度。A * sin(x) → 波高变为 A
  • 频率 F（Frequency）：控制波的密度。sin(F * x) → F 越大波越密
  • 相位 P（Phase）：控制波的偏移。sin(x + P) → P 改变起始位置
Shader 中几乎所有动画的最终形式都是这个公式。
水波纹？用空间距离做 x。
脉动效果？用 u_time 做 x。
变形扭曲？用 sin 的输出去偏移 UV 坐标。
掌握了 A * sin(F * x + P)，你就掌握了 Shader 动画的核心。

在 2D 空间中使用 sin

把 x 替换为 uv.x（水平位置），sin 就变成了横向的波浪条纹。把 sin 的结果当作灰度值，就能看到明暗交替的波纹。

💡 sin 和 cos 的唯一区别
cos(x) = sin(x + π/2) —— cos 只是 sin 向左平移了四分之一周期。它们形状完全一样，只是起始相位不同。所以"需要 cos 还是 sin"只是"从峰值开始还是从零值开始"的区别。

🎯 动手试试

1. 改 frequency 值（试 5.0, 20.0）→ 波纹密度变化

2. 改 amplitude（试 0.1, 0.5）→ 波纹对比度变化

3. 把 sin 换成 cos，观察波纹相位移动了 1/4 周期

4. 用 sin(uv.x * 10.0 + uv.y * 5.0) → 斜向波纹
