🏆 极坐标花朵

📐坐标居中原理⭕SDF 基础🌊sin/cos 波形🎯极坐标系

用极坐标和 cos(θ×N) 创造一朵旋转的花。

目标效果（实时）

花瓣公式

r(θ) = R + A · cos(N · θ)
N = 花瓣数量

💡 提示

  • 极坐标：r = length(uv), θ = atan(uv.y, uv.x)
  • 奇数 N → N 个花瓣，偶数 N → 2N 个花瓣
  • 花心：小半径圆，用暖色（黄色）
  • 加分：让 θ 加上 u_time 让花旋转

🎯 你的任务

1. 画极坐标花瓣

2. 画花心

3. 用角度 θ 给花瓣上色

4. 加分：旋转动画 + 光晕
