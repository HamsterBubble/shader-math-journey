✦ 放射条纹

角度上的"fract"

上节课我们用极坐标画了花瓣。这节课换个思路：不是让 r 跟随 θ 变化，而是只看 θ 本身——把 360° 切成若干等份，就得到了放射状条纹。

关键思路

角度 θ 范围是 -π ~ π（2π 宽）。要分成 N 份：

segmentUV = fract(θ / (2π) * N)

这和之前的空间平铺 fract(uv * N) 完全同理——只是把"空间重复"换成了"角度重复"。

从条纹到图案

  • step(0.5, segmentUV) → 黑白交替的放射条纹
  • smoothstep(0.45, 0.55, segmentUV) → 柔和边缘的条纹
  • 条纹 + 同心环 → 靶心、蜘蛛网、仪表盘刻度
组合极坐标的两个维度：
● 只用 r（距离）→ 同心环：fract(r * 5.0)
● 只用 θ（角度）→ 放射条纹：fract(θ * N / 2π)
● r 和 θ 都用 → 网格、棋盘格、雷达扫描等复杂图案
💡 旋转动画超简单
想让放射图案旋转？只需 θ + u_time。因为极坐标下，角度偏移 = 旋转。不需要矩阵、不需要 sin/cos，一个加法搞定！

🎯 动手试试

1. 改 segments（条纹数 → 3, 8, 16, 32）

2. 把 step 换成 smoothstep → 模糊边缘

3. 叠加同心环：fract(r * 5.0) → 蜘蛛网效果

4. 加旋转动画：angle + u_time * 0.5
