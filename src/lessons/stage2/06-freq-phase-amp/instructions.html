🎚️ 波的三要素：频率、相位、振幅

用音乐类比

把 sin 波想象成一根吉他弦的振动：

  • 振幅 = 弦拨得多用力 → 音量大小
  • 频率 = 弦振动多快 → 音调高低
  • 相位 = 从哪个时刻开始振动 → 起始点偏移

任何波动——无论是声音、光线还是 Shader 中的图案——都完全由这三个参数决定：

y = A × sin(ω × x + φ)

振幅 A (Amplitude)

控制波的"高度"——输出范围变为 [-A, +A]：

  • 0.5 * sin(x) → 波高 ±0.5（温柔的涟漪）
  • 2.0 * sin(x) → 波高 ±2.0（剧烈的起伏）

频率 ω (Frequency)

控制波的"密度"——单位距离内有多少个完整波形：

  • sin(x * 1.0) → 宽阔疏朗
  • sin(x * 10.0) → 密集紧凑
  • 频率翻倍 = 波长减半

相位 φ (Phase)

控制波的"起点"——水平平移整个波形：

  • sin(x + u_time) → 波向左移动
  • sin(x - u_time) → 波向右移动
  • 这就是 Shader 动画的核心：每帧把 u_time 加到相位里
⚡ Shader 动画的秘密
几乎所有动态效果的最终形式都是：
result = amplitude * sin(frequency * position + u_time * speed)
水波、旗帜飘动、草的摇曳、呼吸灯——全是换参数的 sin 动画。
💡 空间变化的参数
参数不必是常数！可以随位置变化：
● 频率随 y 变化：sin((5.0 + uv.y * 10.0) * x) → 上下密度不同
● 振幅随距离衰减：(1.0 / dist) * sin(x) → 波向远处消失
这些"空间变化参数"是创造有机感的关键。

🎯 动手试试

1. 分别改 amp, freq, phase 三个参数观察独立效果

2. 让 freq 随 y 变化: freq = 5.0 + uv.y * 10.0

3. 让 amp 随距离衰减：amp = 1.0 / (1.0 + dist * 3.0)

4. 同时改动三个参数，创造一个"水面"效果
