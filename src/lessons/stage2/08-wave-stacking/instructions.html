🌊 波的叠加原理

一个惊人的事实

1807年，数学家傅里叶证明了一个改变世界的定理：任何复杂的波形，都可以分解为简单 sin 波的叠加。

反过来说：把不同频率、振幅、相位的 sin 波加在一起，就能合成任意波形。这个思想是音频处理、图像压缩（JPEG）、甚至量子力学的基础。

干涉现象

建设性干涉：波峰对波峰

  • 两个相同频率、相同相位的波叠加 → 振幅翻倍
  • sin(x) + sin(x) = 2·sin(x)
  • 现实中：两个音箱播放同一个音，声音变得更响

破坏性干涉：波峰对波谷

  • 两个相同频率、相反相位的波叠加 → 完全抵消
  • sin(x) + sin(x + π) = 0
  • 现实中：降噪耳机的原理！发出反相声波消除噪音

拍频 (Beat)：频率略有不同

  • sin(10x) + sin(11x) → 产生缓慢的"呼吸"效果
  • 两个频率之间的差 = 拍频的频率
  • 在 Shader 中用于创造有机的"脉动"感
方波近似 — 傅里叶级数的经典案例：
用奇次谐波叠加可以逼近方波：
sin(x) + sin(3x)/3 + sin(5x)/5 + sin(7x)/7 + ...
加的项越多，波形越接近完美的方波。这就是傅里叶变换的直觉。
💡 这和噪声有什么关系？
下面的阶段会学 FBM（分形布朗运动）：它就是波的叠加，但每层频率翻倍、振幅减半。
fbm = sin(x) * 1.0 + sin(2x) * 0.5 + sin(4x) * 0.25 + ...
这就是云朵、山脉、火焰纹理的数学基础！

🎯 动手试试

1. 改第二个波的频率，让它接近第一个 → 观察拍频效果

2. 增加更多谐波（5次、7次...）→ 观察趋向方波的过程

3. 用叠加后的波值做高度图着色

4. 试试：sin(x) + sin(2.1*x)*0.5 + sin(3.7*x)*0.25 → 近似随机
