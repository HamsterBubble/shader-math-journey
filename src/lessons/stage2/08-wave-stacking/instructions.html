🌊 波的叠加原理

自然界的复杂波形都是简单 sin 波的叠加（傅里叶分析的核心思想）。

建设性干涉

  • 两个同频率、同相位的波叠加 → 振幅翻倍
  • sin(x) + sin(x) = 2·sin(x)

破坏性干涉

  • 两个同频率、反相位的波叠加 → 相互抵消
  • sin(x) + sin(x + π) = 0

差频拍 (Beat)

  • 两个频率略不同的波叠加 → 产生"呼吸"效果
  • sin(10x) + sin(11x) → 能看到包络线
方波近似：叠加奇次谐波！
sin(x) + sin(3x)/3 + sin(5x)/5 + sin(7x)/7 + ... → 趋近方波

🎯 动手试试

1. 改第二个波的频率, 接近第一个时看"拍频"

2. 增加更多谐波观察趋向方波的过程

3. 用叠加波做高度图着色
