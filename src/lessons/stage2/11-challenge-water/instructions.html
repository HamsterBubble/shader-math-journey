🏆 挑战：水波涟漪

用 sin() 和 length() 创建从中心扩散的水波——Shader 动画中最经典的效果之一。

目标效果（实时）

水波的物理直觉

扔一块石头到水面上，波纹从落点向外扩散。这个过程有两个关键特征：

  • 同心圆：波纹是以落点为中心的圆环 → 用 length(uv - center)
  • 随距离衰减：离落点越远，波纹越弱 → 用 exp(-dist) 或 1/(1+dist)

分步攻略

第一步：基础波纹

wave = sin(distance × frequency - u_time × speed)
  • distance → 圆环形状
  • - u_time * speed → 负号让波向外扩散（正号则向内收缩）
  • frequency → 波纹的疏密

第二步：衰减

  • 线性衰减：wave * max(0.0, 1.0 - dist) → 简单但不自然
  • 指数衰减：wave * exp(-dist * 3.0) → 柔和自然
  • 倒数衰减：wave / (1.0 + dist * 5.0) → 中间区域衰减快、远处慢

第三步：多波源叠加

  • 一个波源太规则。叠加 2-3 个不同位置的波源，互相干涉，效果更自然
  • 可以用 sin(u_time) 让波源缓慢移动
进阶：折射效果
用波纹值去偏移 UV 坐标，再用偏移后的坐标去采样图案：
vec2 distortedUV = uv + wave * 0.02;
这就是"透过水面看物体"的折射效果的基础！
