用三角函数画"美丽曲线"

利萨如图形（Lissajous）是两个不同频率的正弦在 x 和 y 方向叠加产生的曲线。它们出现在示波器、物理振动和音乐可视化中。

x(t) = A · sin(a · t + δ)
y(t) = B · sin(b · t)

参数含义

  • a:b 频率比 — 决定曲线形状。1:1=椭圆，1:2=∞形，2:3=蝴蝶结
  • δ 相位差 — 让曲线"旋转"。δ=0 是直线，δ=π/2 是圆/椭圆
  • A, B 振幅 — 水平和垂直的拉伸

Shader 中如何画参数曲线？

最小距离法：
对每个像素点 uv，遍历参数 t∈[0, 2π]，计算曲线上每个点 p(t) 到 uv 的距离，取最小值。
if (minDist < thickness) → 画线
采样越多曲线越平滑，但越慢。200 个采样点是好的平衡。
💡 glow 效果
除了硬边线条，用 exp(-distance * k) 加一层柔和的光晕，让曲线看起来像霓虹灯管。
