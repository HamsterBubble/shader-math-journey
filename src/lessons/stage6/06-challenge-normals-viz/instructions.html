🏆 挑战：法线可视化

对复杂 SDF 场景计算法线并可视化为颜色 + 光照，创造一个从平面图变成立体感的"神奇一刻"。

目标效果（实时）

分步攻略

第一步：创建 SDF 场景

  • 组合多个 SDF 图元（圆、矩形、线段等）
  • 用 smin/max 做布尔运算

第二步：计算 2D 法线

nx = sdf(p + vec2(ε,0)) - sdf(p - vec2(ε,0))
ny = sdf(p + vec2(0,ε)) - sdf(p - vec2(0,ε))
normal = normalize(vec2(nx, ny))

第三步：光照

  • 把 2D 法线扩展为 3D：vec3(normal, 1.0) 并归一化
  • 对光照方向做 dot 产生漫反射
  • 加 ambient + specular 丰富层次
💡 法线颜色编码
法线分量范围是 [-1, 1]。映射到颜色 [0, 1]：color = normal * 0.5 + 0.5
结果是经典的蓝紫色法线贴图颜色。

🎯 你的任务

1. 创建 SDF 场景（至少 3 个图元）

2. 计算法线并可视化为颜色

3. 加 Lambert 光照让形状看起来"立体"

4. 加分：旋转光源 + Phong 高光
