火焰的视觉分析

火焰有几个关键特征：

  • 形状：底宽顶尖的锥形
  • 颜色：底部亮黄/白 → 中部橙色 → 顶部暗红 → 消失
  • 运动：向上飘动，边缘有随机扰动
  • 明暗：中心最亮，边缘柔和衰减

分步攻略

第一步：基础形状 — 用 SDF 或 UV 遮罩画一个底宽顶尖的区域

第二步：UV 扭曲 — 用向上滚动的噪声扰动边缘
uv.x += noise(uv * freq + vec2(0, -u_time * speed)) * amp

第三步：颜色映射 — 用高度值从白/黄到红/黑渐变
color = mix(vec3(1,0.9,0.3), vec3(0.8,0.1,0), 1.0-intensity)

第四步：亮度衰减 — 顶部和边缘逐渐变暗

简化版噪声（不需要完整 Perlin Noise）：
用 sin(x*a + y*b + t*c) + sin(x*d + y*e + t*f) 叠加几个不同频率的 sin 波就能产生类似噪声的效果。真正的 Perlin/Simplex Noise 是 Stage 7 的内容。
