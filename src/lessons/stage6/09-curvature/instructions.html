曲面的"弯曲程度"

曲率是二阶导数——告诉你函数弯曲的方向和程度。正曲率 = 凹（碗状），负曲率 = 凸（山丘状），零 = 平坦。

κ ≈ f''(x) = (f(x-h) - 2f(x) + f(x+h)) / h²

2D 曲率 = 拉普拉斯

对 2D 标量场，曲率的推广就是拉普拉斯算子：
∇²f = ∂²f/∂x² + ∂²f/∂y²
它衡量一个点和周围的"平均差异"——是扩散方程和热方程的核心。
