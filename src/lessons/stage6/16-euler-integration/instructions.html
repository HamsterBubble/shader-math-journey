🏃 欧拉积分 — 每帧更新速度与位置

✨GPGPU 粒子模拟

为什么需要积分？

导数课学的是「斜率 = 变化率」。真实粒子模拟每帧要做两件事：速度 += 加速度 × Δt，位置 += 速度 × Δt。这就是最简单的 显式欧拉积分。

v(t + Δt) = v(t) + a(t) × Δt
p(t + Δt) = p(t) + v(t) × Δt
本课用 u_time 模拟连续帧：把 sin(u_time) 当作「外力加速度」，在单屏 shader 里画出粒子轨迹圆环，理解「积分 = 累加变化」。
💡 Dracarys 对应
● 火焰粒子：每帧 velocity += gravity，position += velocity
● GPGPU pass：位置纹 / 速度纹双缓冲，与下一课弹簧力模型衔接

🎯 动手试试

1. 设加速度 a = vec2(sin(u_time), cos(u_time)) * 0.5

2. 用 u_time 解析积分出位置 p(t)（圆周运动）

3. 画粒子点 + 速度箭头（用线段或颜色方向暗示）

4. 改阻尼：v *= 0.98 每帧，观察螺旋收敛
