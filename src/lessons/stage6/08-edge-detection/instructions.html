🔍 边缘检测 (Sobel)

梯度大 = 边缘

图像中颜色变化剧烈的地方就是边缘。Sobel 算子用 3×3 采样窗口计算水平和垂直梯度，然后取长度。

Gx = [-1 0 +1; -2 0 +2; -1 0 +1] * image
Gy = [-1 -2 -1; 0 0 0; +1 +2 +1] * image
edge = √(Gx² + Gy²)

Sobel vs 简单差分

  • 简单差分：只采样 2 个点，噪声敏感
  • Sobel：3×3 窗口带加权，平滑+求导同时进行
  • 中心像素权重 2，角落权重 1 → 十字方向敏感度更高
💡 边缘方向
atan(Gy, Gx) 给出边缘的法线方向。结合角度和强度可以做风格化描边。

🎯 动手试试

1. 对 sin 波场做 Sobel 边缘检测

2. 试不同的 epsilon 值

3. 用 Gx/Gy 分量上色 → 边缘方向可视化

4. 反转：1.0 - edge → 漫画描边效果
