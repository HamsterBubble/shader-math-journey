UV 坐标是一切 Shader 图案的起点

所有的 2D Shader 效果，本质上都是对 UV 坐标做某种变换后再"着色"。掌握了 UV 变换，你就掌握了 Shader 图案设计的万能钥匙。

基础 UV 操作清单

  • 平移：uv += offset — 移动图案位置
  • 缩放：uv *= scale — 放大/缩小
  • 旋转：uv = mat2(c,s,-s,c) * uv
  • 镜像：uv = abs(uv) — 沿轴翻转
  • 重复：uv = fract(uv * N) — 平铺
  • 扭曲：uv.x += sin(uv.y * freq) * amp — 波浪形变

组合变换 = 无限可能

每种变换单独都很简单，但组合起来就能产生复杂图案：

  • 平铺 + 旋转 = 旋转的瓦片（Stage 4 学过）
  • 极坐标 + 重复 = 放射状花瓣
  • 扭曲 + 平铺 = 有机纹理
变换的层级思维：
把 UV 变换想象成 Photoshop 的图层——每层做一个简单变换，叠加在一起就是复杂效果。养成"先想好要什么最终效果，再倒推需要哪些变换步骤"的习惯。
💡 修正纵横比
大多数屏幕不是正方形。直接用 UV 画圆会变成椭圆。修正方法：
uv.x *= u_resolution.x / u_resolution.y;
这一行代码让坐标空间变成"等比例"的——画圆就是圆。
