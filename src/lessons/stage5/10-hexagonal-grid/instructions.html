⬡ 六角网格

📈smoothstep 原理⭕SDF 基础

蜂巢最优结构

正方形网格用 fract 就行，但六角网格需要偏移行的技巧。六角形是最紧密的铺地方式——蜜蜂早就知道了。

核心算法

s = vec2(1.0, √3) // 六角格子尺寸
p1 = mod(uv, s) - s/2 // 偶数行中心
p2 = mod(uv - s/2, s) - s/2 // 奇数行中心
d = min(length(p1), length(p2)) // 最近中心距离

应用

  • 蜂巢纹理、六角瓷砖、战棋地图
  • 六角 Bokeh（景深虚化）
  • 六角网格+噪声 = 自然的裂纹图案
