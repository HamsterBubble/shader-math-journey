SDF 复习与扩展

Stage 1 初识了 SDF（有符号距离场）。现在我们来建立一个完整的 2D SDF 基础形状库——这些形状是所有复杂图形的积木。

核心形状公式

  • 圆：length(p) - radius
  • 矩形：max(abs(p.x) - w, abs(p.y) - h)——更精确的版本用 length(max(abs(p) - size, 0.0))
  • 等边三边形：用 dot 和 abs 组合
  • 线段：用投影和 clamp 求点到线段的距离
  • 正多边形：用 atan + cos 的周期性

SDF 的符号含义

SDF 值的三种状态：
● d < 0 → 在形状内部
● d = 0 → 在形状边界上
● d > 0 → 在形状外部

符号信息至关重要——它让我们可以区分内外，做布尔运算（合并/相交/挖洞）。

从 SDF 到颜色

SDF 只返回一个距离值。要变成画面，需要"着色"：

  • 硬边：step(0.0, -d)
  • 柔边：smoothstep(0.01, -0.01, d)
  • 轮廓：smoothstep(0.02, 0.0, abs(d))
  • 光晕：exp(-abs(d) * 20.0)
