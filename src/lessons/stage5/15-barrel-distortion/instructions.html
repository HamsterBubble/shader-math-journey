🔭桶形畸变 Barrel Distortion

镜头感从哪来？

沿径向推拉 UV，模拟广角镜头边缘弯曲。Dracarys Master pass 在慢动作时加强畸变，强化电影镜头感。

公式

cc = uv − 0.5
d = dot(cc, cc)
uv' = uv + cc × d × strength
采样顺序：先算畸变后的 UV，再用它读取棋盘/场景颜色——这是后处理 pass 的标准写法。

🎯 动手试试

1. 实现 barrel(uv, strength) 返回畸变 UV

2. 用畸变 UV 采样棋盘 checker(uv)

3. 左半原图、右半畸变对比（或全屏畸变）

4. 调 strength（正 = 桶形，负 = 枕形）
