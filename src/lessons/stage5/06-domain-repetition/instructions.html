♾️ 域重复：一个形状 → 无限复制

域重复 (Domain Repetition) 是 SDF 最强大的技巧之一：

repeated_uv = mod(uv + period/2, period) - period/2

  • 只定义一个形状的 SDF，自动获得无限重复
  • 比 for 循环高效无数倍
  • 配合 floor() 获取每个副本的 ID（给不同颜色/旋转）

限制域重复

clamp(id, -N, N) → 只在指定范围内重复

⚡ Ray Marching 中的域重复可以用一个球的 SDF 创造出无限的3D空间——这是很多 Shadertoy 作品的核心技术。
