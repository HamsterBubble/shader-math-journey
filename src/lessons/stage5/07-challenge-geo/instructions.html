🏆 挑战：几何图案海报

综合运用 SDF 图元、布尔运算、域重复和变换矩阵，创造一幅印刷级几何图案。

目标效果（实时）

设计思路

好的几何图案通常有层次和节奏：

  • 大结构：整体构图（中心放射 / 网格 / 对角线）
  • 中细节：每个区域内的图案变化
  • 小装饰：点、线条、光晕等微小元素增加精致感

推荐工具箱

  • SDF 图元 + 布尔运算 → 自定义形状
  • 洋葱化 + 轮廓 → 线框风格
  • 域重复 + cellID 差异 → 无限且不单调
  • IQ 调色板/HSV → 丰富的色彩
  • 极坐标 → 放射对称
💡 "减法"设计哲学
与其从零开始堆砌，不如从一个铺满的图案开始，然后用 max（差集）"雕刻"出最终形状。就像雕塑家从一块大理石上凿出作品。
