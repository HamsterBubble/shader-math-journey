🔗 SDF 布尔运算组合

从 Stage 1 的 min/max 到真正的 CSG

Stage 1 学过 min/max 可以做形状的合并和相交。现在我们深入学习完整的 SDF 布尔操作体系。

三种基本布尔运算

Union（并集）= min(d1, d2)
Intersection（交集）= max(d1, d2)
Subtraction（差集）= max(d1, -d2)

平滑布尔运算 (Smooth CSG)

硬 min/max 的问题是接缝处有锐利的尖角。在自然界中，物体的融合是圆滑的。平滑版本让形状"融化"在一起：

平滑并集 (smin)：

smin(a, b, k) — k 控制融合程度
  • k = 0 → 硬 min（完全锐利）
  • k = 0.1 → 微微圆滑
  • k = 0.5 → 显著融合（Metaball 效果）

最常用的 smin 实现：

float smin(float a, float b, float k) {
  float h = clamp(0.5 + 0.5*(b-a)/k, 0.0, 1.0);
  return mix(b, a, h) - k*h*(1.0-h);
}
Metaball（融球）效果：
把多个圆的 SDF 用 smin 合并，当两个球靠近时，它们会像水滴一样"粘"在一起。
这个效果在 Shader 社区极受欢迎——只需 smin + 几个移动的圆就能创造沸腾的岩浆般的视觉。
