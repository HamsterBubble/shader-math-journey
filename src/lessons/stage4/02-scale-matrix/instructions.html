缩放矩阵

缩放矩阵是最简单的变换矩阵——对角线上放缩放倍数：

| sx   0 |
| 0   sy |
  • sx = sy = 2.0 → 均匀放大 2 倍（UV 坐标放大 = 图案看起来缩小！）
  • sx = 2.0, sy = 0.5 → x 方向拉伸，y 方向压缩
  • sx = -1.0 → 水平镜像翻转

变换组合：顺序决定一切

矩阵乘法不可交换：A × B ≠ B × A。这意味着变换的顺序至关重要。

旋转圆 vs 旋转椭圆：
● 先缩放再旋转 = 圆被压扁成椭圆后整体旋转
● 先旋转再缩放 = 圆先旋转后再被拉伸
同样的两个变换，顺序不同结果完全不同。在代码中：
uv = scaleMat * rotateMat * uv → 先旋转（右侧先应用），后缩放
💡 "反向思维"提醒
在 Shader 中，变换是施加在坐标上的。所以：
● 放大 UV → 图案看起来缩小
● 顺时针旋转 UV → 图案逆时针旋转
这和游戏引擎中变换物体是反过来的。
