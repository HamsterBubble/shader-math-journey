🏆 挑战：跳动的心形

用 SDF 和矩阵缩放画一个有节奏跳动的心形——Shader 领域最浪漫的练习。

目标效果（实时）

心形 SDF 的两种方法

方法 A：数学公式法

d = x² + (y - √|x|)² - r²

这个公式用 √|x| 来"撑起"上半部分的两个弧，下半部分自然形成尖角。简单但优雅。

方法 B：几何组合法

  • 画一个 45° 旋转的正方形（SDF = max(abs(x-y), abs(x+y))）
  • 在正方形的两条斜边上各放一个半圆
  • 用 min 合并 → 心形

跳动动画

心脏跳动不是简单的 sin——它有"快速收缩、慢速舒张"的节奏：

  • 基础：scale = 1.0 + 0.1 * sin(u_time * 6.0)
  • 更真实：scale = 1.0 + 0.15 * pow(sin(u_time * 4.0), 2.0)
  • 用 pow 让波峰更尖锐 → 模拟心跳的"砰-砰"节奏
💡 呼吸光晕
在心形外围加一层 exp(-dist * k) 光晕，让 k 也随心跳节奏变化 → 心脏"发光"的效果。光晕颜色可以比心形本体稍浅、稍暖。
