矩阵是什么？

矩阵是一种"坐标变换机器"——输入一个 vec2，输出另一个 vec2。mat2 是 2×2 矩阵，能做旋转、缩放、剪切等变换。

旋转矩阵推导

旋转角度 θ 后，原来在 (1,0) 的点跑到了 (cos θ, sin θ)，原来在 (0,1) 的点跑到了 (-sin θ, cos θ)。把这两个新基向量排成矩阵：

| cos θ   -sin θ |
| sin θ    cos θ |

GLSL 中的 mat2

GLSL 的 mat2 写法有个坑：它是按列填充的！

  • mat2(a, b, c, d) → 第一列 (a,b)，第二列 (c,d)
  • 旋转矩阵：mat2(cos(θ), sin(θ), -sin(θ), cos(θ))
  • 使用：vec2 rotated = rotMat * uv;
⚠️ 列优先 vs 行优先
数学教材写矩阵是按行读的，但 GLSL 的 mat2/mat3/mat4 构造函数是按列填充。忘记这个会导致旋转方向反转！
旋转 UV 和旋转物体是反的：
在 Shader 中我们旋转的是"坐标网格"，不是"物体"。把 UV 顺时针旋转 30°，图案看起来会逆时针旋转 30°——就像转地图和转你自己的区别。
💡 封装旋转函数
实际项目中常写一个辅助函数：
mat2 rot2d(float a) { float c=cos(a), s=sin(a); return mat2(c,s,-s,c); }
一行搞定，后面直接 uv *= rot2d(angle);
