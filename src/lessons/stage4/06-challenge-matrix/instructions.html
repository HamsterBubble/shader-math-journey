万花筒的数学原理

真实的万花筒用镜子反射来创造对称。在 Shader 中，abs() 就是"镜子"——它把负半边翻转到正半边。

分步攻略

核心循环：旋转 → 折叠 → 旋转 → 折叠

  1. 旋转坐标 π/N（N 是对称数）
  2. uv = abs(uv) — 沿 x 和 y 轴做镜像
  3. 重复 2-4 次

每次 abs 折叠都会把空间"对折"一次。折叠 N 次后，空间被分成 2^N 个对称区域。在最终的"折叠空间"里画一个简单图案，它会被展开成复杂的对称花纹。

对称数与旋转角度的关系：
● 6 重对称 → 旋转 π/3（60°）→ 雪花、六角形
● 8 重对称 → 旋转 π/4（45°）→ 八角星
● 旋转角度 = π / (对称数 / 2)
在 Shadertoy 搜索 "kaleidoscope" 会找到上千个作品。
💡 颜色也可以被折叠！
万花筒不只限于形状。你可以先画一幅彩色图案，然后对 UV 做折叠，"对称"会自动应用到颜色上，产生宝石般的迷幻效果。
