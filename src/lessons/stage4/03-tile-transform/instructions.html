🧩 每个格子独立变换

网格 + 矩阵 = 无限可能

前面我们用 fract() 创建了重复网格，用 mat2 做了旋转。现在把它们组合起来——让每个格子都有自己独立的变换。

关键思路

  1. vec2 cellID = floor(uv * N); → 获取格子编号
  2. vec2 cellUV = fract(uv * N) - 0.5; → 每格局部坐标，中心在 (0,0)
  3. 用 cellID 计算每格独有的变换参数（旋转角度、缩放比例...）
  4. 对 cellUV 施加变换后画图案

为什么先减 0.5？

旋转和缩放都是围绕原点进行的。如果不减 0.5，"格子的原点"在左下角，旋转效果会很奇怪。减去 0.5 把原点移到格子中心，旋转就自然了。

伪随机差异化：
用 cellID 生成伪随机值，让每格看起来不同：
float rand = fract(sin(dot(cellID, vec2(12.9898, 78.233))) * 43758.5453);
● 旋转角度 = rand * 6.28
● 缩放比例 = 0.5 + rand * 0.5
● 颜色 = 基于 rand 的调色板
💡 交替行为
mod(cellID.x + cellID.y, 2.0) 可以让相邻格子交替（棋盘格模式）。用它控制旋转方向：偶数格顺时针，奇数格逆时针 → 齿轮咬合效果！
