距离不只有一种

在日常生活中，"距离"就是两点间的直线长度。但在数学和 Shader 中，"距离"的定义可以有很多种——而不同的定义会产生完全不同的形状。

欧几里得距离 → 圆形

d = √(x² + y²) = length(p)

这就是我们熟悉的"直线距离"（勾股定理）。等距线是圆形。

曼哈顿距离 → 菱形

d = |x| + |y|

想象在网格状的城市里走路——只能沿东西南北走，不能走对角线。这就是"城市街区距离"。等距线是45° 旋转的正方形（菱形）。

切比雪夫距离 → 正方形

d = max(|x|, |y|)

想象国际象棋中国王的走法——每步可以走任意方向一格。走 N 步能到达的范围是一个 N×N 的正方形。等距线是正方形。

Minkowski 距离 → 形状过渡

d = (|x|ⁿ + |y|ⁿ)^(1/n)
  • n = 1 → 曼哈顿（菱形）
  • n = 2 → 欧几里得（圆形）
  • n → ∞ → 切比雪夫（正方形）
  • n = 0.5 → "超级星形"（向外凸的十字）

调节 n 值，你可以在菱形、圆形、方形之间平滑过渡！

距离类型是 SDF 图案设计的核心技巧：
只需切换距离函数，同一套代码就能生成截然不同的图案。圆形网格 → 菱形网格 → 方形网格，只需一行改动。
