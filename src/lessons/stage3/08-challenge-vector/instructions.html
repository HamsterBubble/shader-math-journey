🏆 挑战：带高光的球体

用本阶段的向量数学知识画一个有漫反射 + 高光的 2D 球体——这是 3D 图形学的"Hello World"。

目标效果（实时）

分步攻略

第一步：构建 3D 法线

我们在 2D 平面上模拟 3D 球体。球体表面的法线可以从 UV 坐标推算：

  • 球面上的点：vec3(p.x, p.y, sqrt(r² - p.x² - p.y²))
  • 归一化后就是法线 N（表面朝向）
  • 当 x² + y² > r² 时，像素在球外 → 丢弃

第二步：漫反射（Lambert）

diffuse = max(dot(N, L), 0.0)

N = 法线，L = 光照方向。越面对光源越亮。

第三步：高光（Phong）

R = reflect(-L, N)
specular = pow(max(dot(R, V), 0.0), shininess)

V 通常取 vec3(0, 0, 1)（垂直屏幕向外），代表观察者的视线方向。

第四步：合成

  • color = ambient + diffuse * materialColor + specular * lightColor
  • ambient = 环境光（避免完全黑），通常 0.1 ~ 0.2
💡 菲涅尔效应（加分项）
真实物体的边缘比正面更亮（因为反射率随角度增加）。简单近似：
float fresnel = pow(1.0 - dot(N, V), 3.0);
把 fresnel 加到边缘，球体立刻有了"通透"感。
