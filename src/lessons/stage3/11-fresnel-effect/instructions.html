"边缘发光"的物理原理

观察水面、玻璃球、气泡——边缘总是比中心更亮。这就是菲涅尔效应：光线以掠射角（接近平行）入射时，反射率急剧增加。

fresnel = (1 - dot(N, V))^n
N = 表面法线, V = 视线方向, n = 锐度（通常 2-5）

直觉理解

  • 中心：法线朝向你 → dot(N,V) ≈ 1 → fresnel ≈ 0（不发光）
  • 边缘：法线垂直于视线 → dot(N,V) ≈ 0 → fresnel ≈ 1（强烈发光）
  • 指数 n：n 越大，发光区域越集中在边缘

2D 中模拟菲涅尔

简化方法：
对于一个 2D 圆，"法线"就是 normalize(uv)。"视线"近似为 -normalize(uv)。
所以 dot(N, V) = 1 - length(uv)/R，在圆心=1，边缘=0。
直接用 pow(1.0 - NdotV, n) 就是菲涅尔。
💡 Schlick 近似
真实物理的菲涅尔公式很复杂。Schlick 近似 F₀ + (1-F₀)(1-cosθ)⁵ 是游戏行业标准。
F₀ = 0.04（电介质）或 0.7-1.0（金属）。
