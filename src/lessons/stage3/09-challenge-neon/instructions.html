用向量距离场和 exp() 衰减创造赛博朋克风格的霓虹发光效果。

目标效果（实时）

霓虹光的数学原理

霓虹灯的特征是：线条本身非常亮，周围有柔和的光晕逐渐衰减。这正好是 exp(-distance) 的行为！

glow = intensity / distance
或 glow = exp(-distance × sharpness)

分步攻略

第一步：画基础形状的 SDF

  • 线段 SDF：用向量投影算点到线段的距离
  • 圆环 SDF：abs(length(p) - radius)
  • 可以组合多个基础形状

第二步：转换为发光效果

  • 不用 step/smoothstep 画硬边，而是用衰减函数
  • glow = 0.01 / dist → 简单的 1/x 衰减，中心极亮
  • glow = exp(-dist * 20.0) → 可控的指数衰减

第三步：上色

  • color += glowIntensity * neonColor
  • 不同形状用不同颜色（粉色、青色、紫色常见于赛博朋克）
  • 颜色相加（不是 mix）→ 重叠区域更亮

第四步：闪烁动感

  • 微弱闪烁：glow *= 0.9 + 0.1 * sin(u_time * speed)
  • 不同线条用不同闪烁频率 → 更有生命力
配色建议：
● 粉色：vec3(1.0, 0.2, 0.6)
● 青蓝：vec3(0.2, 0.8, 1.0)
● 电紫：vec3(0.6, 0.2, 1.0)
● 背景用深暖色 vec3(0.02, 0.01, 0.03) 增强对比

🎯 你的任务

1. 画 2-3 条发光线条或圆环

2. 用 exp 或 1/x 做辉光衰减

3. 不同线条使用不同霓虹颜色

4. 加分：让线条缓慢移动或脉动
