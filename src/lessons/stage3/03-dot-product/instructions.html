💡 dot() — 点积

Shader 中最重要的运算

如果只能学一个向量运算，那一定是点积。它是光照、阴影、反射、投影……几乎所有 3D 效果的数学基础。

代数定义

dot(a, b) = a.x × b.x + a.y × b.y（+ a.z × b.z ）

就是对应分量相乘再相加。结果是一个标量（数字，不是向量）。

几何含义：测量"相似度"

dot(a, b) = |a| × |b| × cos(θ)

当 a 和 b 都是单位向量（长度=1）时，点积直接等于cos(夹角)：

  • dot = 1 → 完全同方向（θ = 0°）
  • dot = 0 → 完全垂直（θ = 90°）
  • dot = -1 → 完全反方向（θ = 180°）
  • dot = 0.5 → 夹角 60°

简而言之：点积衡量两个方向有多"对齐"。

光照：Shader 的灵魂公式

Lambert 漫反射：
float light = max(dot(normal, lightDir), 0.0);

● normal = 表面朝向（单位向量）
● lightDir = 光照方向（单位向量）
● 面朝光源（dot ≈ 1）→ 亮
● 侧面（dot ≈ 0）→ 暗
● 背对光源（dot < 0）→ 完全黑，用 max 截断

这一行代码就实现了 80% 的真实感光照效果。
💡 dot 的其他用途
● 求向量长度的平方：dot(v, v) = |v|²（比 length 快，避免 sqrt）
● 投影：dot(v, axis) = v 在 axis 方向上的分量长度
● 角度判断：dot > 0 = 前方，dot < 0 = 后方
