💡 dot() — 点积

几何含义

dot(a,b) = |a|·|b|·cos(θ)

对于单位向量：dot = cos(夹角)

  • dot = 1 → 同方向
  • dot = 0 → 垂直
  • dot = -1 → 反方向
Shader 最重要的公式：
float light = max(dot(normal, lightDir), 0.0);
面朝光源的面亮，背对光源的面暗。

🎯 动手试试

1. 改 lightDir 方向（试各种 vec2）

2. 用鼠标控制光源方向（提示：uniform u_mouse）

3. 把 max(..., 0.0) 去掉，看负值区域
