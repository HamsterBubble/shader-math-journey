什么是向量？

在 Shader 中，向量无处不在。vec2 可以表示：位置、方向、速度、UV 坐标、力……它们的数学操作完全相同。向量就是一组有序的数，Shader 对它们做逐分量运算。

向量加法 = 位移/偏移

  • vec2(1,2) + vec2(3,4) = vec2(4,6) — 每个分量分别相加
  • 几何含义：从 A 出发，沿着 B 的方向和距离移动
  • 例如：uv + vec2(0.1, 0.0) → 整个画面向左移动 0.1（为什么是左？因为坐标增大 = 采样点右移 = 图案看起来左移）

向量减法 = 求"从A到B的箭头"

  • vec2(5,3) - vec2(2,1) = vec2(3,2)
  • 几何含义：从点 A 指向点 B 的向量
  • uv - center → 从 center 指向 uv 的向量，包含了方向和距离两个信息
这是最常见的 Shader 模式之一：
vec2 p = uv - center;
一行代码同时得到：
● length(p) → 到中心的距离（用来画圆、做衰减）
● normalize(p) → 方向（用来做放射效果、光照）
● p.x, p.y → 相对位置（用来做极坐标变换）
⚠️ 加法的方向易混淆：uv + offset 移动的是"采样位置"，图案看起来是反方向移动的。这和 CSS 的 translate 方向一致，但初学者常常搞反。
