🔄 矩阵变换

用矩阵乘法实现旋转、缩放、剪切 — GPU 的原生思维方式。

一、为什么用矩阵？

矩阵的威力：一次乘法 = 同时完成旋转+缩放+剪切。而且多个变换可以预先合并成一个矩阵。

二、2D 旋转矩阵

mat2 rotate = mat2(cos(θ), -sin(θ),
                   sin(θ), cos(θ));
vec2 rotated = rotate * uv;

四、变换速查

变换mat2 构成GLSL
旋转 θ[cosθ, -sinθ, sinθ, cosθ]mat2(c,-s,s,c)
均匀缩放 s[s, 0, 0, s]mat2(s,0,0,s)
X 剪切 k[1, k, 0, 1]mat2(1,0,k,1)
反射 (X轴)[1, 0, 0, -1]mat2(1,0,0,-1)
💡 GLSL 的 mat2 是列优先！
mat2(a, b, c, d) 表示：
第 1 列 = (a, b)，第 2 列 = (c, d)
所以旋转写作 mat2(cos,-sin, sin,cos)
注意 -sin 在第一列第二位，而非右上角！
💡 变换的顺序很重要
rotate * (scale * uv) 先缩放后旋转
scale * (rotate * uv) 先旋转后缩放
两者结果不同！
变换 = 矩阵 × 坐标
旋转用 cos/sin | 缩放用对角线 | 组合用矩阵乘法 | 列优先！
