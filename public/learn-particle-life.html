💫 粒子生命周期 — 从诞生到熄灭

火焰、烟雾、火花不能永远存在。GPGPU 粒子的 Alpha 通道存生命值 life，每帧递减；归零则重生——形成连续的「起—燃—熄」循环。

一、life 通道：粒子的「倒计时」

float life = texture2D(uPosTex, uv).a; · 范围通常 [0, 1] · 1 = 刚诞生 · 0 = 该重生了

与位置/速度存在同一张纹理里，更新 pass 一次读写全部状态，无需 CPU 介入。

二、更新逻辑：衰减 + 重生

每帧递减

life -= uDecayRate * dt;
if (life <= 0.0) {
  life = 1.0;                                    // 重置生命
  pos  = randomPointOnEmitterSurface();          // 随机发射点
  vel  = uInitialVelocity + randomJitter();        // 初速度 + 扰动
}

渲染 pass：life 驱动视觉

float age = 1.0 - life;                          // 0=年轻 1=将死
vec3 color = mix(uHotColor, uColdColor, age);    // 黄→红→灰
float size = mix(uMaxSize, uMinSize, age);       // 大→小
float alpha = life * smoothstep(0.0, 0.1, life); // 末尾淡出
