🔁 fract 取小数与图案重复

只保留小数部分 — 将无限空间切割成无限重复的格子。

一、函数定义

fract(x) = x - floor(x)

效果：无论 x 多大，输出永远在 [0, 1) 之间循环。

二、数值直觉

xfloor(x)fract(x)理解
0.300.3小数部分
1.710.7去掉整数
3.1430.14只留小数
5.050.0整数 → 0
-0.3-10.7负数也安全

三、图形效果：锯齿波

y = x（线性）

y = fract(x)（重复）

四、交互：控制密度

4

使用 fract(uv * N)，N 越大 → 格子越多

五、2D 空间重复

vec2 cell_uv = fract(uv * 4.0) — 将画面均匀切成 4×4 格子，每个格子内 UV 从 0~1 重新开始。

六、与 floor 搭配

💡 fract + floor = 完整信息
vec2 cellId = floor(uv * N); — 哪个格子 (整数 ID)
vec2 cellUv = fract(uv * N); — 格子内坐标 (0~1)
两者配合 → 同时知道"在哪个格子"和"格子内的位置"
💡 偏移形成交错
if (mod(floor(uv.y * N), 2.0) == 1.0) uv.x += 0.5;
奇数行偏移半格 → 砖墙排列效果
fract(x) = 永远在 0~1 循环
乘 N 再 fract → N 倍重复 | floor = 格子序号 | fract = 格子内坐标
