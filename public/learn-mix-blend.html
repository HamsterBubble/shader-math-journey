🎨 mix — Shader 的万能调色板

mix() 是 Shader 中使用频率最高的函数之一。它的本质就一句话：按比例混合两个值。不管是颜色、位置、大小还是形状，全都能混。

一、像调奶茶一样理解 mix

mix(a, b, t) = a × (1 - t) + b × t

想象你面前有两杯饮料：A 杯是纯咖啡 ☕，B 杯是纯牛奶 🥛。t 就是你往杯子里倒了多少比例的牛奶：

  • t = 0.0：一滴牛奶都没放 → 纯咖啡（100% A）
  • t = 0.3：放了三成牛奶 → 偏咖啡色的拿铁
  • t = 0.5：咖啡和牛奶对半 → 标准拿铁
  • t = 1.0：杯子里全是牛奶 → 纯牛奶（100% B）

在 Shader 中，A 和 B 可以是任何东西：两个颜色、两个位置、两个亮度数字……而 t 就是"偏向 B 的程度"。

三、数值速查表

0.0■ 纯红一滴 B 都没有，100% 全是 A

四、Shader 中最常见的四大用法

🎨 用法 ① 上色（SDF 的标配）

vec3 color = mix(背景色, 前景色, circle_mask);

圆的 SDF 遮罩是 0（外部）或 1（内部），带有平滑过渡边缘。把它当做 t 传进 mix，那些 0 的地方自动显示背景，1 的地方变成前景色。抗锯齿边缘则**按比例自动混合**两种颜色！

🌅 用法 ② 天空渐变

vec3 color = mix(地面棕, 天空蓝, uv.y);

uv.y 从 0(底部) 到 1(顶部)。屏幕下方 t≈0 显示棕色地面，上方 t≈1 变成蓝天，中间自然过渡。一行代码画出日落渐变！

⏱ 用法 ③ 时间动画

float t = smoothstep(0.0, 1.0, u_time);
vec3 color = mix(状态A, 状态B, t);

让时间 u_time 驱动 t 从 0 慢慢爬向 1，画面就在两个状态之间平滑过渡。这就是 Shader 动画的核心手法！

📐 用法 ④ 不止是颜色！

mix 可以混合任何类型的值：

  • float size = mix(0.1, 0.5, pulse); → 大小在 0.1 和 0.5 之间随 pulse 变化
  • vec2 pos = mix(posA, posB, t); → 两个位置之间的平滑移动（做动画！）
