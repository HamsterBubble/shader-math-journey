🎨 mix 颜色混合与插值

Shader 中最常用的颜色混合工具 — 线性插值的本质。

一、函数定义

mix(a, b, t) = a × (1 - t) + b × t

当 t = 0 时返回 a，当 t = 1 时返回 b，中间值则按比例混合。

二、数值速查

tmix(红, 蓝, t)含义
0.0纯红100% a, 0% b
0.25偏红75% a, 25% b
0.50中间紫50-50 对半
0.75偏蓝25% a, 75% b
1.0纯蓝0% a, 100% b

四、在 Shader 中的典型用法

🎨 SDF 着色（最常用）
vec3 color = mix(bg, fg, circle_mask);
circle_mask 为 0 → 背景色，为 1 → 前景色，边缘处平滑过渡。
🌅 渐变
vec3 color = mix(color1, color2, uv.y);
UV 的 y 从 0→1，颜色自然从 color1 过渡到 color2。
⏱ 动画过渡
float t = smoothstep(0.0, 1.0, u_time);
vec3 color = mix(stateA, stateB, t);
用时间驱动 t，实现状态之间的平滑动画。
💡 mix 不限于颜色
可以混合任何类型：float、vec2、vec3、vec4
float size = mix(0.1, 0.5, pulse); — 混合数值
vec2 pos = mix(posA, posB, t); — 混合位置
t = 0 → 全 a ，t = 1 → 全 b
mix = 线性插值 | t 来自 smoothstep → 平滑过渡 | t 来自 UV → 渐变
