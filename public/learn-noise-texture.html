🌫️ 噪声纹理 — 预烘焙的随机性

Shader 里实时算 Perlin/FBM 噪声功能强大但昂贵。预先把噪声烘焙成纹理，运行时只需 texture2D(noiseTex, uv + time)——便宜、稳定，是全屏氛围效果的利器。

一、为什么用纹理而不是纯数学？

纹理采样的优势

  • 性能：一次 texture2D vs 多层 sin/cos/dot 运算
  • 可控：美术可手绘或离线生成特定风格的噪声图
  • 多八度：FBM 可以预叠在 RGB 各通道，或分层多张图
  • 可平铺：Repeat 包裹实现无缝滚动雾效

Dracarys 使用 perlin.png、noise-fbm.png、fognoise.jpg 等预烘焙资源。

二、典型采样模式

// 滚动雾效
float fog = texture2D(uFogNoise, uv * 2.0 + vec2(time * 0.02, 0.0)).r;

// 多层叠加（不同尺度 + 速度）
float n1 = texture2D(uNoise, uv * 3.0 + time * 0.01).r;
float n2 = texture2D(uNoise, uv * 8.0 - time * 0.02).r;
float density = n1 * 0.6 + n2 * 0.4;

// 距离衰减 × 噪声
float fade = 1.0 - smoothstep(0.3, 1.0, length(uv - 0.5));
color += vec3(fog) * fade * 0.5;

四、Dracarys 环境氛围（dracarys-08-atmosphere）

雾 + 闪烁 + 距离衰减

案例 Master Pass 组合多层噪声纹理，配合到龙/相机的距离做 fade，营造体积感而不需真实体积渲染。鼠标轨迹 noise 扰动则来自 NDC 坐标驱动的 UV 偏移。

vec2 noiseUV = uv * uNoiseScale + vec2(uTime * 0.015, uTime * 0.008);
float n = texture2D(uFogNoise, noiseUV).r;
float distFade = 1.0 - smoothstep(uFogNear, uFogFar, depth);
atmosphere = n * distFade * uFogDensity;
