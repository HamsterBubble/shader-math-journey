🗺️ 法线贴图 — 用 RGB 伪造微观凹凸

真实水面波纹需要百万级顶点。法线贴图把微观表面朝向编码进 RGB 纹理，Shader 采样后偏移反射 UV——零额外几何，却有丰富的镜面细节。

一、法线贴图里存的是什么？

每个 texel 的 RGB 代表该点切线空间法线的方向向量：

vec3 normal = texture2D(uNormalMap, uv).rgb * 2.0 - 1.0;
纹理值 [0,1] → 法线 [-1,1] · 默认 (0.5, 0.5, 1.0) = 平坦朝上
  • R → X 分量（左右倾斜）
  • G → Y 分量（前后倾斜）
  • B → Z 分量（朝向表面外，通常最亮）

二、扰动反射 UV 的三步

① 采样法线并解码

vec3 n = texture2D(uNormalMap, uv).rgb * 2.0 - 1.0;

② 用 XY 分量偏移采样坐标

vec2 reflectUV = baseUV + n.xy * uStrength;

法线往右倾 → 反射图像往右「滑」一点，模拟凹凸造成的折射/反射偏移。

③ 用扰动后的 UV 采样环境/场景

vec3 reflection = texture2D(uScene, reflectUV).rgb;

三、交互对比：有无扰动

拖动「扰动强度」滑块。左：平面反射（无扰动）；右：法线贴图扰动后的反射。

0.035

左：flat UV · 右：uv + normal.xy × strength

四、Dracarys 水面反射（dracarys-04-water-reflect）

案例从波动纹理的梯度推导法线，再扰动天空盒/场景采样 UV。龙飞过水面时，反射会随波纹实时扭曲——这是法线贴图课与波动方程课的综合应用。
