〰️ 波动方程 — 让水面活起来

真实流体模拟代价极高。游戏和创意案例常用简化离散波动方程：每个像素的高度由四邻域平均决定，配合速度积分形成涟漪扩散。

一、物理直觉：弹簧网格

想象每个像素是一根小弹簧，四邻域拉它回到「平均水平」。偏离越大，回弹力越强——这就是离散波动的本质。

高度 h ≈ 水面位置 · 速度 vel ≈ 高度变化率 · 每帧交替更新

二、核心公式（每帧每个像素）

更新规则

float avg = (north + south + east + west) * 0.5;
float newH  = avg - prevVel;       // 四邻平均 − 旧速度
float newVel = newH - oldH;        // 新速度 = 高度差分
// 可选阻尼
newVel *= 0.995;
newH   *= 0.998;

这不是精确 Navier-Stokes，但视觉上足够 convincing，且 GPU 友好。

鼠标激励源

float dist = length(uv - mouseUV);
float bump = cos(dist * 40.0) * exp(-dist * 8.0);
newH += bump * clickStrength;

点击时叠加 cos 相位鼓包，模拟石子入水的涟漪源。
