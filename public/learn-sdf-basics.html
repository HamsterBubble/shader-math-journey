⭕ SDF 有符号距离函数

Shader 中画图形的核心方法 — 用距离值描述几何形状。

一、核心概念

SDF = Signed Distance Function。它对空间中的每个点返回一个数：

d < 0

内部

d = 0

边界

d > 0

外部

"有符号"指的就是：负数 = 内部，正数 = 外部。一个数字同时编码了"在哪边"和"离多远"。

二、圆的 SDF（最简单的例子）

sdf_circle(p) = length(p - center) - radius

length(p - center) 是点 p 到圆心的距离，减去半径后：

位置length(p)- radius结果
圆心 (0,0)0.0- 0.4-0.4 (内部)
边界上0.4- 0.40.0 (边界)
远离圆心0.8- 0.4+0.4 (外部)

五、SDF 的超能力

💡 布尔运算
• 并集（多个形状）= min(sdf1, sdf2)
• 交集（重叠区域）= max(sdf1, sdf2)
• 差集（挖洞）= max(sdf1, -sdf2)
💡 形状变形
• 描边 = abs(sdf) - thickness
• 圆角 = sdf - roundness
• 膨胀 = sdf - offset
SDF = 到边界的"带方向的距离"
负 = 内部 | 零 = 边界 | 正 = 外部 | smoothstep 出颜色 | min/max 做布尔
