📈 smoothstep — Shader 里的"万能柔化剂"

如果 step() 是一把冰冷的裁纸刀，那 smoothstep() 就是一支温柔的毛笔。它能把任何锐利的"非黑即白"二值断裂，变成一道自然的缓坡过渡。

一、先用生活比喻理解它

想象你在做一碗热巧克力奶：

  • step() 就像一瞬间把整碗巧克力粉倒进牛奶——从纯白到纯棕色，中间没有任何过渡，分界线清清楚楚！
  • smoothstep() 则是你拿着勺子慢慢搅拌——巧克力粉从碗的边缘缓缓向中心弥散，最终变成丝滑的均匀渐变色。

在 Shader 里，这种丝滑的渐变就是抗锯齿和柔和过渡的核心武器。

二、三步拆解它的内部机器

smoothstep(edge0, edge1, x)

这个函数内部其实干了三件事，我们逐个拆开：

机器零件 ①：线性归一化

t = (x - edge0) / (edge1 - edge0)

这一步的白话意思是："x 走到了 edge0 到 edge1 这段路的百分之几？"

  • x 还没到起点 edge0？那 t 是负数（还没上路）→ 被钳制为 0
  • x 正好在起点 edge0？t = 0（刚出发）
  • x 走到了一半 (edge0+edge1)/2？t = 0.5（走了一半）
  • x 到达终点 edge1？t = 1.0（旅程结束）
  • x 远远超过终点？被钳制为 1（已经走完了）

机器零件 ②：安全阀 clamp

t = clamp(t, 0.0, 1.0)

无论 x 输入多离谱（负一百万或正一百万），t 都被死死夹在 [0, 1] 之间。这保证了输出不会越界，永远安全。

机器零件 ③：Hermite 魔法曲线

result = t² × (3 - 2t)

这是整个 smoothstep 的灵魂！如果只是直线 t，得到的就是一条普通的直线斜坡（像 mix）。但走这条 S 形曲线后，起步慢、中间快、结束也慢，就像汽车起步要加速、到站要刹车。这种"软着陆"让画面看起来极其自然！

三、亲眼看 step vs smoothstep vs 线性

三种过渡曲线并排对比。注意红色（step）的暴力一刀切、绿色（线性）的直线对角、以及蓝色（smoothstep）那条优雅的S弯：

■ step — 一刀切 ■ 线性 — 直线斜坡 ■ smoothstep — S曲线

六、Shader 实战中的三大金牌用法

用法 ① 抗锯齿圆（SDF 的事实标准搭档）

float circle = smoothstep(0.01, -0.01, d);

SDF 得到的距离 d 在边界附近从正到负。用一个极窄的 smoothstep 在 d=0 附近做过渡，圆的边缘就不再是像素级别的锯齿！

用法 ② 用相减画线（脉冲函数）

float line = smoothstep(A, B, x) - smoothstep(B, C, x);

两个 smoothstep 相减，就能精确地从画面中雕刻出一条带有天然柔和边缘的细线。这就是课程五中那条金色分割线的原理！

用法 ③ 倒置 = 渐隐

float fade = smoothstep(0.7, 0.3, x);

当 edge0 > edge1 时，效果完全翻转！从 1 到 0，实现渐隐消失。
