📈 smoothstep 平滑阶跃

Shader 中使用频率最高的函数之一，用于创建平滑过渡。

一、函数签名

float smoothstep(float edge0, float edge1, float x)

返回 0.0 ~ 1.0 之间的值，根据 x 相对于 [edge0, edge1] 区间的位置做平滑插值。

二、底层数学

smoothstep 内部做了三步：

t = clamp((x - edge0) / (edge1 - edge0), 0.0, 1.0)
result = t × t × (3 - 2t)
步骤作用输出范围
① 线性映射(x - edge0) / (edge1 - edge0)-∞ ~ +∞
② clamp限制到 [0, 1]0 ~ 1
③ Hermite 插值t²(3 - 2t)，起点终点斜率都是 00 ~ 1 (平滑)

三、对比：step vs smoothstep

step(edge, x)

硬切：x < edge → 0，x ≥ edge → 1

smoothstep(e0, e1, x)

平滑过渡：在 [e0, e1] 之间渐变

六、常用技巧

💡 反转 smoothstep
smoothstep(0.7, 0.3, x) — 当 edge0 > edge1 时，结果从 1 → 0
效果：在 x = 0.3 处值为 1，x = 0.7 处值为 0。常用于制作渐隐效果。
💡 窄边 = 硬边（模拟 step）
smoothstep(0.49, 0.51, x) — edge 间距极小时，效果接近 step，但有微小抗锯齿。
这是 SDF 渲染中最常用的抗锯齿技巧！
💡 制作条纹/环
smoothstep(0.3, 0.31, x) - smoothstep(0.5, 0.51, x)
两个 smoothstep 相减 = 一个精确的条带选择器。
edge0 = 开始过渡，edge1 = 完成过渡
区间越窄 → 越锐利 | 区间越宽 → 越柔和 | 反转 → 渐隐
