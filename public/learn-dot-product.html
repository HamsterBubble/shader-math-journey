🔘 dot 点积运算

衡量两个向量"方向一致性"的核心工具 — 光照计算的基石。

一、数学定义

dot(A, B) = A.x × B.x + A.y × B.y
= |A| × |B| × cos(θ)

两个公式等价。第一行是计算方法，第二行揭示几何意义：结果与两向量夹角 θ 的余弦成正比。

二、夹角 → 值

同向 (0°)

dot = +1

垂直 (90°)

dot = 0

反向 (180°)

dot = -1

（以上值基于两个单位向量 |A| = |B| = 1）

三、交互可视化

🖱 移动鼠标改变蓝色向量方向，观察 dot 值变化：

dot = 0.00
角度 = 90°
关系 = 垂直

四、Shader 中的用法

场景代码原理
漫反射光照max(0., dot(N, L))法线与光线越对齐 → 越亮
菲涅尔反射1.0 - dot(N, V)视线越平行边缘 → 反射越强
投影长度dot(p, axis)点在轴上的投影
方向判断dot(dir1, dir2) > 0.0> 0 同侧, < 0 对侧
💡 dot(v,v) = length²(v)
向量和自身点积 = 长度的平方。比 length(v) 快（避免 sqrt），常用于性能优化。
dot = 方向一致性 = cos(夹角)
