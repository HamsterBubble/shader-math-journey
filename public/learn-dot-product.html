🔘 dot 点积："两个向量有多一致？"

点积(dot product)是整个 Shader 光照系统的心脏。它本质上回答了一个简单的问题：两个方向是不是朝着同一边走？

一、先用手电筒理解

你拿着一支手电筒🔦照向一面墙壁🧱。手电筒有一个照射方向，墙壁有一个法线方向（"正脸朝哪"）：

  • 手电筒正对墙壁（两个方向完全平行且互相朝向）→ 墙壁被照得最亮 → dot = 1.0
  • 手电筒与墙壁呈90° 直角（光几乎擦边而过）→ 墙壁几乎没光 → dot = 0.0
  • 手电筒在墙壁背面（光朝反方向打）→ 完全没有光 → dot = -1.0（负数！）

所以 dot() 输出的就是一个 -1 到 1 之间的"一致性评分"。越接近 1 越同向，越接近 -1 越反向，0 就是完全无关。

二、数学其实超级简单

dot(A, B) = A.x × B.x + A.y × B.y

就是把两个向量的 x 分量相乘、y 分量相乘，然后加起来。三维空间再加一个 z。

如果两个向量都是单位长度（归一化后 length = 1），那么 dot 的结果就等于两个方向之间夹角的余弦值 cos(θ)。这就是它被用来做光照的原因！

夹角 θcos(θ) = dot含义Shader 用途
0°1.0完全同向正面光照最强
45°0.707略微偏斜稍弱但还算明亮
90°0.0垂直擦边侧面光/掠射边缘
135°-0.707偏向背面几乎无光
180°-1.0完全反向背面完全黑暗

四、Shader 中的典型用法

☀️ 用法 ① 朗伯光照 (Lambertian Diffuse)

float brightness = max(0.0, dot(normal, lightDir));

表面法线和光线方向越一致（dot 越大），表面越亮。用 max(0, ...) 防止背面产生负数亮度。这三行就是所有 3D 游戏光照的原点！

🎭 用法 ② 边缘光 (Fresnel / Rim Light)

float rim = 1.0 - dot(normal, viewDir);

当视线与表面越接近平行（dot 趋近 0），1-dot 趋近 1，边缘就越亮。这就是轮廓光/菲涅尔效应的秘密！

🗺️ 用法 ③ 方向选择（半球采样）

float mask = step(0.0, dot(normal, direction));

dot > 0 意味着"这个方向面朝我"，可以用来决定哪些面应该被渲染、哪些应该被剔除。
