💡 SDF 辉光：为什么 exp(-abs(d)) 会发光？

第 5 课画圆的最后一行 color += fg * 0.15 * exp(-abs(d) * 15.0) 给圆加了一圈光晕。它把 SDF 距离 d 翻译成离边界越远、光越弱的物理衰减。

一、两行代码各干什么？

vec3 color = mix(bg, fg, circle);           // ① 实心圆本体
color += fg * 0.15 * exp(-abs(d) * 15.0);   // ② 叠加辉光层
  • ① mix：用 circle 蒙版在背景和前景色之间插值，画出实心圆
  • ② +=：在已有颜色上叠加额外亮度（加法发光，见 光斑与色彩算术）
  • 0.15：辉光强度系数，越大越亮
  • 15.0：衰减速度，越大光晕越窄、越锐利

二、exp(-x) — 自然界最爱的衰减

exp(-x) 当 x=0 时为 1（最亮），x 越大越接近 0（熄灭）。光的散射、雾气的浓度、电阻的放电都近似这种曲线。

glow = exp(-abs(d) * k)
把 abs(d) 当作「到边界的距离」，距离 0 → 亮度 1，距离越远 → 指数级变暗。

三、关键：为什么用 abs(d)？

SDF 的 d 有符号：圆内为负，圆外为正，边界为 0。abs(d) 去掉符号，变成「到边界的距离」——不管在里还是在外，只要离边界同样远，亮度就一样。

位置dabs(d)exp(-abs(d)×15)视觉效果
圆心-0.40.4≈ 0.002⬛ 很暗
靠内边缘-0.020.02≈ 0.74🟦 亮
边界001.0💡 最亮！
靠外边缘+0.020.02≈ 0.74🟦 亮
远离圆外+0.40.4≈ 0.002⬛ 很暗

所以最亮点在圆的轮廓上，而不是圆心。圆心离边界远，反而暗——这就是游戏里的 Rim Glow（边缘光），适合能量护盾、传送门、霓虹描边。

四、三种写法 = 三种发光风格

① exp(-abs(d) * k) — 边缘双向辉光

边界最亮，向内向外对称衰减。圆心偏暗，外圈有光晕。本课默认写法。

② exp(-max(0.0, d) * k) — 纯外发光

圆内 d < 0 被 max 截成 0，exp(0)=1 恒亮 → 内部均匀饱和，只有外部有光晕。适合发光球、荧光灯。

③ exp(-d * k)（不加 abs）— 核心发光

圆内 d 为负 → -d 为正 → 越靠近圆心 exp 越大，能量向中心暴涨；圆外快速熄灭。适合太阳、恒星核心。

五、调参指南

  • 0.15（强度）：改成 0.3 光晕更刺眼，改成 0.05 更含蓄
  • 15.0（衰减 k）：改成 5 光晕铺得更开、更柔和；改成 40 只剩极细的边缘亮线
  • 想要「只有外圈发光」→ 换 max(0.0, d)
  • 想要「实心发光球」→ 本体用 mix 填色 + 外发光用 max(0.0,d) 叠加
辉光 = 距离场 + 指数衰减 + 颜色叠加 (+=)。弄懂这三个零件，霓虹灯、力场、魔法阵都能同一套公式拼出来。
