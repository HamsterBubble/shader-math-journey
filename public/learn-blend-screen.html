💡 Screen 滤色混合 — 让光叠加更亮

普通 mix(a, b, 0.5) 会把两色平均变暗。Screen（滤色）混合则模拟两束光叠在一起——结果永远比任一输入更亮，是火焰、光晕、高光的标配。

一、公式：双重反相再相乘

screen(a, b) = 1 - (1 - a) * (1 - b)

// GLSL 向量版
vec3 screen(vec3 a, vec3 b) {
  return 1.0 - (1.0 - a) * (1.0 - b);
}

直觉：先各自「还剩多少暗」(1-a)，暗部相乘后更暗，再反回来 → 亮部保留并增强。

二、与 Normal / Add / Multiply 对比

四、Dracarys 后处理（dracarys-07-post-master）

光晕 + 龙焰叠加

vec3 scene = texture2D(uScene, uv).rgb;
vec3 glow  = texture2D(uBloom, uv).rgb;
vec3 flame = texture2D(uFlame, uv).rgb;

vec3 result = screen(scene, glow);    // 先叠 Bloom
result = screen(result, flame);       // 再叠火焰
// Screen 保证光效自然变亮而不轻易 clip 到 1.0
