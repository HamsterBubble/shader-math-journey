❓ 代码彻底拆解：画一个会发光的圆

这个案例终于把我们带进了图形学殿堂级的大门：SDF (有符号距离场)。在这里我们将看到坐标系变形、反向剔除、以及辉光物理衰减的奥秘。

第一步：重置坐标系原点（折纸居中法）

1. uv = uv * 2.0 - 1.0;

2. uv.x *= u_resolution.x / u_resolution.y;

在 Shader 默认的规矩里，长方形屏幕左下角是(0,0)，右上角是(1,1)。要画一个**位于正中心的圆**就会非常麻烦。所以这起手的两句话成了无数特效的核心起手式：

  • uv * 2 - 1：把 0~1 的座标，瞬间撑成了 -1 到 1。于是，绝对的正中心 0 面板跑到了屏幕**最中间**，像极了初中数学的十字坐标系。
  • uv.x *= 屏幕宽高比：大部分屏幕都是长方形的（比如 16:9）。这会导致你算出的圆变成“被拉扯的扁胖椭圆”。乘以宽高比，相当于把 X 轴重新压缩回了物理等比例状态。

第二步：一把测距用的虚空圆规 (SDF 概念)

float d = length(uv) - radius;

什么是 length(uv)？它是获取坐标点 到原点(0,0)的直线距离！

👉 为什么叫“距离场”？

假设我们的半径 radius 给定了 0.4。我们跑遍整个屏幕，看看每个点距离圆心的距离差值（距离值减去 0.4）：

  • 点 A 距离中心是 0.1：d = 0.1 - 0.4 = -0.3。它的结果是负数，说明这个点深陷在目标圆内。
  • 点 B 距离中心正好是 0.4：d = 0.4 - 0.4 = 0.0。说明点踩在了边界线上！
  • 点 C 距离中心是 0.8：d = 0.8 - 0.4 = 0.4。结果是正数，说明在圆外。

这就是大名鼎鼎的 SDF：负数为内，0为壳，正数为外！有了距离 d，我们就能任意描边甚至做外发光！

第三步：反向思维画出实体 (smoothstep 倒置技巧)

float circle = smoothstep(0.01, -0.01, d);

看到这你可能会疑惑：smoothstep 不是从小到大（例如 0.4 到 0.5）去产生一个 0~1 的渐变平滑过渡吗？为什么会给 0.01, -0.01 这样的大到小参数？

巧妙的逆转裁切

当 smoothstep(A, B, x) 中 A 大于 B 时，它的效果会完全翻转：

  • 如果点在圆外很远 (d > 0.01，正数)：输出 0 (黑色/背景)。
  • 如果点在圆内部深处 (d < -0.01，负数)：输出 1 (纯白色实体)！
  • 如果正好踩在圆的边界线附近 (-0.01 到 0.01)：给它一点平滑过渡，这叫抗锯齿(Anti-Aliasing)。

只需要这一句话，距离场就变成了一个实心的图形蒙版！

第四步：核能发光衰减公式 (exp指数函数)

color += fg * 0.15 * exp(-abs(d) * 15.0);

我们在之前的课程学过了发光就是 += 的混合原理。现在的问题是发光需要离光源越近越亮，离开越远光消散越快。这叫光的物理衰减特性。exp(-x) 就是自然界中最完美的衰减器。

🔥 关键疑问：为什么 abs(d) 导致圆内部也发光？

这是一个非常敏锐的观察。让我们把这个问题彻底想透：

🧠 abs() 的真实效果：双向对称衰减

d 是 SDF 距离：圆外为正数，圆内为负数，边界为 0。

加上 abs() 之后，不管你在圆里面还是外面，距离统统变成正数。这意味着衰减函数 exp(-abs(d)*15) 变成了一座以圆边界为中心、向内外双向对称的"山脊"：

像素位置 d 值 abs(d) exp 结果 亮度
圆心正中 -0.4 0.4 ≈ 0.002 ⬛ 极暗
靠近内边缘 -0.02 0.02 ≈ 0.74 🟦 很亮
正好在边界 0.0 0.0 1.0 💡 最亮峰值！
靠近外边缘 +0.02 0.02 ≈ 0.74 🟦 很亮
远离圆外 +0.4 0.4 ≈ 0.002 ⬛ 极暗

所以辉光的最亮点不在圆心，而在圆的边界轮廓上！圆心因为离边界太远（abs(d) 很大），辉光反而衰减掉了。这就是游戏里常见的 "边缘发光 / Rim Glow" 效果。

💡 三种写法 = 三种截然不同的发光风格

根据你对 d 的处理方式不同，同一个 exp 衰减函数会产生完全不同的视觉效果：

  • exp(-abs(d) * 15.0) → 边缘双向辉光 (Rim Glow)：圆的边界最亮，向内向外都衰减。圆心反而偏暗。适合做能量护盾、传送门光环。
  • exp(-max(0.0, d) * 15.0) → 纯外发光 (Outer Glow)：圆内部 d<0 被 max 截为 0，exp(0)=1 恒亮。内部完全均匀饱和，只有外部边缘有辉光。适合做发光球体、荧光灯。
  • exp(-d * 15.0)（不加 abs 也不 max）→ 内亮外灭 (Core Glow)：圆内 d<0 变成 exp(正数)，越往圆心能量越猛烈暴涨！圆外则极速消失。适合做太阳、恒星核心。
