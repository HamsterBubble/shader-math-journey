🌀 极坐标 — 用"距离+角度"重新看世界

在直角坐标系(x, y)中，每个点的身份证号码是"向右走多少、向上走多少"。但在极坐标中，身份证变成了"离中心多远(r)、在哪个钟点方向(θ)"。这种坐标系对于画旋涡、放射线、花瓣等圆形对称图案有着统治级的优势。

一、从直角坐标到极坐标

float r = length(uv); → 到中心的距离

float theta = atan(uv.y, uv.x); → 从 3 点钟方向逆时针转了多少角度

就像你站在一个圆形大厅的正中间 🏛️，有人问你："小明在哪？"

  • 直角坐标的回答："往东走3米，往北走4米"→ (3, 4)
  • 极坐标的回答："离你5米远，在你的1点钟方向"→ (r=5, θ=53°)

两种回答描述的是同一个位置！只是看问题的方式不同。

三、极坐标的超能力

🌊 超能力 ① 放射线只需一行

在直角坐标中画放射状线条超级复杂。但在极坐标中：

float rays = sin(theta * 8.0);

一行代码！因为 θ 天然地绕着中心扫一圈，sin(θ*N) 就产生了 N 条等分的放射线。

🌸 超能力 ② 花瓣/齿轮/星形

圆的 SDF 是 r - radius。如果让 radius 随 θ 变化呢？

float radius = 0.3 + 0.1 * sin(theta * 5.0);

半径随角度呈正弦波抖动 → 5 片花瓣！改 5 为 6 就是 6 角星。这就是极坐标的魅力——角度就是天然的形状参数。

🌀 超能力 ③ 旋涡

float spiral = fract(r * 5.0 - theta / 6.28);

让 r 和 θ 互相纠缠！距离越远角度偏移也越大，于是螺旋线自然诞生。

四、坐标转换速查

方向公式GLSL 写法
直角 → 极坐标r = √(x²+y²)
θ = arctan(y/x)		r = length(uv);
theta = atan(uv.y, uv.x);
极坐标 → 直角x = r × cos(θ)
y = r × sin(θ)		uv = vec2(cos(theta), sin(theta)) * r;

注意 atan(y, x) 的取值范围是 [-π, π]（约 -3.14 到 3.14）。如果想映射到 [0, 1]，可以做 (theta / 6.2832) + 0.5。
