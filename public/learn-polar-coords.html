🎯 极坐标系

用角度 + 距离描述位置 — 径向图案和旋转的神器。

一、从笛卡尔到极坐标

笛卡尔 (x, y) → 极坐标 (r, θ)

r = length(uv) = √(x² + y²)
θ = atan(y, x)   // 范围 [-π, π]

r 是到原点的距离，θ 是与 X 轴正方向的夹角。

三、对比两种坐标

笛卡尔坐标 (x, y)

✅ 方格、条纹
✅ 水平/垂直运动
❌ 圆形不方便

极坐标 (r, θ)

✅ 圆、扇形、螺旋
✅ 径向渐变/重复
❌ 直线不方便

四、GLSL 写法

操作代码效果
求角度float angle = atan(uv.y, uv.x);-π ~ π
求距离float r = length(uv);0 ~ ∞
极→笛vec2 p = vec2(cos(θ), sin(θ)) * r;还原回 xy

五、典型应用

🌸 花朵图案
r = cos(θ * N) — N 控制花瓣数量
🌀 螺旋
fract(θ/(2π) - r * twist) — 角度与距离混合产生螺旋
🎨 径向渐变/分区
color = palette(θ / (2*π)) — 用角度驱动颜色，形成色轮
r = length(uv)，θ = atan(y, x)
距离 r → 圆环效果 | 角度 θ → 扇形/花瓣 | 混合 → 螺旋
