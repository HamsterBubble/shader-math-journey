🔭 桶形畸变 — 模拟广角镜头的边缘弯曲

真实相机镜头尤其是广角镜，画面边缘会向中心「鼓出」。后处理中用径向 UV 偏移模拟这种桶形畸变（Barrel Distortion），能瞬间赋予画面电影镜头感。

一、径向推拉：离中心越远，偏移越大

以画面中心 (0.5, 0.5) 为原点，计算每个像素到中心的距离平方，沿径向方向推拉 UV：

vec2 cc = uv - 0.5;
float d = dot(cc, cc);           // 距离²，避免 sqrt
uv = uv + cc * d * strength;     // strength > 0 → 桶形（边缘外鼓）

strength > 0 桶形畸变（广角感）· strength < 0 枕形畸变（边缘内收）

二、为什么用 dot(cc, cc) 而不是 length？

性能与效果

  • dot(cc, cc) = r²，单调递增，偏移趋势与 r 一致
  • 省一次 sqrt，全屏后处理每像素都受益
  • 高阶版本可用 d + d² * k2 + d³ * k3（Brown-Conrady 模型）

四、Dracarys Master Pass（dracarys-07-post-master）

案例在后处理 Master Pass 中根据慢动作进度动态加强桶形畸变——时间减速时镜头感骤然增强，配合暗角和色差，营造「龙息凝滞」的电影瞬间。

float barrel = mix(0.0, uMaxBarrel, slowMoFactor);
vec2 cc = uv - 0.5;
uv += cc * dot(cc, cc) * barrel;
vec3 color = texture2D(uScene, uv).rgb;
