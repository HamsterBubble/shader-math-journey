✨ 颜色加减与光效叠加：Shader底层原理

在 WebGL 和 Shader 开发中，我们绝不能用现实中“刷油漆覆盖”的思维去理解颜色。屏幕背后的本质是受激发光的二极管，所谓的色彩合成，底层就是严酷的数字加减乘除。

只要弄懂这几套数学法则，Photoshop里的“滤色、叠加、正片叠底”以及游戏里的“激光、极光、冲击波”你都能靠几行代码徒手捏出来！

第一部分：四大基础色彩算法（光之四则运算）

我们设定：底层是一团蓝色星云 A = `vec3(0, 0, 0.8)`；飞来了一发红色激光 B = `vec3(0.8, 0, 0)`。两团像素发生相撞时的四种基础反应：

➕ 加法 ( += )：极亮发光叠加

底层逻辑：光子越聚越多，能量直接相加。
公式: result = A + B。超出 1.0 的部分会溢出变成刺眼的死白。

特效应用：镭射波、强光束、霓虹灯、魔法特效。这是一切自发光材质的绝对基石。在图像软件中称为 Linear Dodge (线性减淡)。

vec3 result = A + B; // -> vec3(0.8, 0, 0.8) 洋红色，非常亮

✖️ 乘法 ( *= )：遮罩过滤 / 衰减

底层逻辑：由于 RGB 是 [0, 1] 之间的小数，乘法等于打折（越乘越暗）。任何光乘黑(0)都会被无情抹掉。

特效应用：阴影贴花、根据 SDF 创建的遮罩裁剪、全局降低亮度。在图像软件中这叫做 Multiply (正片叠底)。

vec3 result = A * B; // -> vec3(0,0,0) 全黑！纯蓝与纯红没有交集

⚖️ 插值 mix(A, B, alpha)：阿尔法混合

底层逻辑：像放一张半透明的贴纸。由 Alpha (0~1) 控制 B 借用多少 A 的面积。
公式: A*(1 - a) + B*a

特效应用：渲染带透明度的 UI、画硬边缘的几何体盖在底图上。这是经典的 Normal (正常混合)。

vec3 result = mix(A, B, 1.0); // -> vec3(0.8,0,0) A 完全被 B 盖掉

➖ 减法 ( -= )：凿洞与色相反转

底层逻辑：抽干光子。
公式: result = A - B，结果低于 0.0 底限则被截断为彻底的深渊黑。

特效应用：在发光的网格中劈开一道黑暗的裂缝、用 1.0 - color 制作负片反相效果。在软件里称为 Subtract (减去)。

vec3 result = A - B; // -> 蓝光被红光减，互不相关，依然是蓝色

第三部分：高阶特效公式方案（PS神技复刻）

纯粹的相加或相乘有时候会过于极端（太亮或太暗）。Shader 前辈们根据人眼感光曲线，推导出了很多极其优美的改良公式。

✨ 方案一：Screen (滤色) — 最完美的柔和发光光晕

直接用 A+B 相加的话很容易突破1.0产生死白，细节会全部糊掉。滤色(Screen) 是发光特效中最通用的防爆白方案。它会让底层和顶层的光像投屏机一样叠加，永远不会超过纯白，非常适合做极光、星空光晕。

// Screen 公式 (A, B 必须是 0.0 ~ 1.0 之间)
vec3 screenEffect(vec3 A, vec3 B) {
    return 1.0 - (1.0 - A) * (1.0 - B);
}

✨ 方案二：Overlay (叠加) — 神秘的对比度增强

做游戏场景后期或材质强化时无敌好用。如果底层偏亮，它就在上面做滤色发光；如果底层偏暗，它就在上面做正片叠底。它会让亮的更亮，暗的更黑！

// Overlay 公式 (需要依据底色 A 的亮度分情况计算)
vec3 overlayEffect(vec3 A, vec3 B) {
    // 这里是一个快速拟合近似的极简写法
    // 根据底色 A 决定使用哪种混色倾向
    vec3 result;
    for(int i=0; i<3; i++) { // 对RGB分别计算
        if(A[i] < 0.5) result[i] = 2.0 * A[i] * B[i];
        else result[i] = 1.0 - 2.0 * (1.0 - A[i]) * (1.0 - B[i]);
    }
    return result;
}

✨ 方案三：Soft Light (柔光) — 注入氛围色

就像在镜头前蒙上一层有颜色的玻璃纸。如果你写了一个极度黑暗冷酷的灰度迷宫 Shader，可以在最后一步加上一点暖橘色的 Soft Light，绝不破坏原有图案的立体感。

// 简化的柔光模拟公式
vec3 softLight(vec3 A, vec3 B) {
    return A * (1.0 - A) * B + A;
}
